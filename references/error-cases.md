# 实战错误案例库（持续累积）

本文件累积**真实写作中暴露的文风/设定问题**，作为技能的经验库。每条案例都来自实际项目反馈，包含「现象 → 根因 → 错误写法 → 正确写法 → 可固化规则」，供后续创作前对照规避。案例中的人名、地名、专有机制均为**脱敏后的通用占位**（主角 / 某角色 / 某镇 / 某宗门 / 某符号），便于套用到任意作品。

> **如何新增案例**：在遇到新的、可复用的文风/设定问题时，复制文末「案例模板」追加到本文件，编号顺延（#001、#002…）。每条力求简短（错误/正确片段各 ≤150 字），便于快速扫描。

> **⚠️ 机检闭环纪律（最高优先，来自 2026-07-17 ch8–11 复检教训）**：每次重写/修改正文后，必须对本章跑**一遍覆盖全部可机检模式的完整扫描**，不能只跑「字数 + 说·道 + 禁词 + 相邻重复」这四项基础 QC。本次 ch8–11 复检即因缺全扫，漏掉 ch10 整章 136 个 ASCII 英文引号（自检报告还假称「引号统一「」」）与 7 处 #023「一X」突发词残留。可机检模式至少含：破折号数（≤3）、引号规范（本书用「」/『』，禁 ASCII 引号）、一X/名词一动词突发词、X了一分族、否定式（不X不Y/像在X/防御性不是X）、前世类词、旧名残留。各项目应自备扫描脚本（如 scan_errors.py），写后必跑，假阳性手动甄别，真违规即修。

## 案例 #001 — 故弄玄虚式文风（系统性，最致命）

- **来源**：用户长篇连载项目反馈（2026 年初，前 5 章）
- **现象（用户原话）**：「少读几段时没有感觉，读多了就感觉故弄玄虚，不好好说话，很难一直读下去」
- **根因（三个叠加，非单章硬伤）**：
  1. **信息空转**：同一信息换种说法反复讲（如某角色进门「银灰线+残念」原样说两遍；章尾「远方的线」连说 5 次；「某角色古纹能感应」绕 4 遍）。读者潜意识捕捉到「故事原地打转」便烦躁。
  2. **章尾无限抒情拖尾**：正戏结束后又接 20+ 行意象堆叠感慨，每章如此，读者每次到章尾都得熬一段作者独白。
  3. **文言虚词密度过高**：通篇端着，没有一句大白话做缓冲，像全程拿腔拿调。
- **错误写法（片段）**：
  > 腕上灰线忽地一跳——这年轻人腕上也缠着一道线，尾端泛着极淡的银灰，像从千万里外一路缠来……识海里的残念，极轻地动了一息，似认得那银灰的来路，却又不全认得。（同一信息在第 5、9 行原样复述）
- **正确写法（片段）**：
  > 主角抬眼打量，腕上灰线忽然跳了一下。这年轻人腕上也缠着一道线，尾端泛着极淡的银灰，像从很远的地方缠过来。识海里那点残念动了动，仿佛认得，又不全认得，只留下个模糊的「远」字。（信息只说一次）
- **可固化规则（三删三加）**：
  - **删复读**：一个点只说一次，关键处最多呼应一次。
  - **删章尾抒情拖尾**：正戏结束即收，压到 2–3 句，落在动作或钩子上，不铺意象。
  - **删空转虚词**：删「半晌/足见/端的」等无信息量虚词。
  - **加平实缓冲句**：文言腔之间插入大白话短句，松紧交替。
  - **加说人话的对话**：对话带急、顿、噎，别都「不疾不徐」。
  - **加具体动作细节**：用动作/感官代替抽象感慨。

## 案例 #002 — 设定一致性硬伤（主角出身门派却「听说过」）

- **来源**：用户长篇连载项目反馈（中段章节）
- **现象**：原文写「主角早听过某宗门的名」，但角色卡设定主角原身是该宗门**外门散修**——自己出身的门派不可能用陌生口吻「听说过」。
- **根因**：写前未核对角色卡/世界观中的出身设定，凭惯性用了「旁观者」视角描述。
- **错误写法**：
  > 主角早听过某宗门的名。域内正道宗门里排得上号……
- **正确写法**：
  > 某宗门这名号，主角不陌生。原身未离山那会儿，他就是那宗门外门散修，后离山流落某镇。这门守契的名声域内数得着……
- **可固化规则**：写前必须核对角色卡出身/势力；主角对自己出身的门派、地名，**不应用陌生口吻描述**，改用「旧识/本就出自」类表述。

## 案例 #003 — 跨章专名不统一（镇 A vs 镇 B）

- **来源**：用户长篇连载项目反馈（前 5 章）
- **现象**：同一镇子在前后章出现两个名字；且其中一个与某宗门名混淆（宗门在山门上，与镇名无关）。
- **根因**：专名未建立 canon 清单，章节间各自发挥。
- **可固化规则**：开写前建立「专名 canon 清单」（镇名、宗门名、势力名、主角出身），全书严格统一。例：主舞台镇=镇 A；宗门=宗门 X（在山脉 Y）；主角=该宗门外门散修后流落镇 A。跨章改动专名后必须 grep 全卷清零旧写法。

## 案例 #004 — 别扭称呼（用错身份标签指主角）

- **来源**：用户长篇连载项目反馈（中段章节，全章 4 处同类误称）
- **现象**：主角本是散修出身，却被称作「外乡修士」，读者觉别扭。
- **根因**：称呼未与角色出身对齐。
- **错误写法**：「何须寻我一个外乡修士？」「区区外乡人，凭什么……」
- **正确写法**：「何须寻我一个散修？」「外人罢了，凭什么……」
- **可固化规则**：称呼/自称须与角色身份一致；散修出身者不称「外乡」，改用「散修/外人」。

## 案例 #005 — 看不懂的因果句（词义模糊 + 因果不清）

- **来源**：用户长篇连载项目反馈（中段章节）
- **现象**：「今年某宗门下脉三个弟子卡在炼气瓶颈，半数是灵气不足所致」——「下脉」词义不明，整句因果读者读不通。
- **根因**：用了作品内部黑话（下脉）且未交代因果链条。
- **错误写法**：「今年某宗门下脉三个弟子卡在炼气瓶颈，半数是灵气不足所致。」
- **正确写法**：「今年某宗门下游一脉，有三个弟子卡在炼气瓶颈上——其中两个，分明是灵气亏虚拖的，寻常年景哪会这样。」
- **可固化规则**：避免作品内部黑话；因果句须把「因→果」说清，模糊处加一句解释或落到具体现象。

## 案例 #006 — 设定符号过度复读（核心视觉化符号每次出场重建）

- **来源**：用户长篇连载项目反馈（全书，2026-07-11）
- **现象**：某核心视觉符号（文中以「线」具象化因果，以下用「线」作通用示例）在全稿出现 600+ 次，单章 24–66 次，读多了心烦。四类复读：①跨章重描（已出场角色的符号新章又完整描）②章内累进复读（第 N 个新符号把前 N-1 个的「引/感/承」全量复读）③固定系统反应（每次见新符号系统必动）④意象反复（「远方/大网」跨章复读）。
- **根因**：把核心设定的视觉化符号当成**每次出场都要重建**的信息，而非「首建、后续仅状态变化」。
- **错误写法（某章某角色金线段，同章内复读）**：
  > 第9行：这姑娘腕上也缠着一道线——清亮得像初春的溪，尾端浮着金…系统残念没动…连某角色甲那道银灰残念都认得个「远」字… 第17行：某角色甲银灰是「引」、某角色乙古纹是「感」、这姑娘金线是「承」，三道线三种力道，来自远方，比某反派那枚印大得多的网。
- **正确写法**：
  > 这姑娘腕上也缠着一道线，尾端浮着极淡的金——和前头见过的线都不是一路。主角心头微异，残念竟拿不准这金线的来路。（首次建立即可；后续该角色再出场只提「那道金线」；不回述引/感/承全貌）
- **可固化规则（设定符号降频纪律，详见 style-guide 第六节）**：
  - 首描详、复描略；不写累进对比段。
  - 系统/残念反应只在关键剧情点触发，非每次见符号都动。
  - 大意象（远方/大网）一次定性，不再抒情复读。
  - 破案必需的符号推理保留，只删四类复读冗余。
  - **用户二次反馈（关键修正）**：换字/减字频只是治标——读者烦的不是某个字多，而是**关于该符号的叙事节拍（看符号/想符号/符号亮）调用太密**。正确做法是**减少概念被调用的叙事节拍数**：删不推动剧情的冗余调用（章尾「符号亮了一息」、meta 总结「看因果一日比一日清楚」、情绪点「摸着那道亮符号」、多案密集罗列「看符号」），把空缺用案情/人物/世界生态填回。减字频（线→痕/因果）只能辅助，不能替代减节拍。

## 案例 #007 — 超自然感知「直白贴标签」（减了节拍仍出戏）

- **来源**：用户长篇连载项目反馈（2026-07-12）
- **现象（用户原话）**：「还是有点多，应该说是写的太直白了，你可以描写主角的动作，比如主角看了手腕，但是完全不提线和痕」
- **根因**：减了冗余节拍、换了字，但凡写到「感知」仍直接把超自然机制**标出来**（「灰痕一跳」「某红线连着某人腕上」「金痕拿不准」）。读者从首章已知主角有某种「看见」之能，后续无需每次点名机制；直接命名=唠叨且出戏。
- **错误写法（某章重做片段）**：
  > 主角一进堂，腕上灰痕便微微发沉——他看见，长老周身缠着一道灰红，自心脉起缓缓蚀着神识，那灰红的尾端，恰恰连在师姐腕上。
- **正确写法（动作+反应，不命名机制）**：
  > 主角的目光在长老和那师姐之间走了一遭。长老手抖得厉害，师姐递汤药的手却稳得很——稳得不像照料病人，倒像在数着日子。主角心里，那根弦无声地绷了一下。
- **可固化规则**：
  - 超自然感知**首章建立一次**后，后续章用「动作 + 对方可读取的神色 + 主角结论」传递，不重复命名机制符号（线/痕/灰红/金痕/古纹/银灰等）。
  - 写「他看了眼手腕」「凑近标本」「盯着手」「把腕子缩回袖子」等物理动作，让读者自行补足「他在用那双眼睛看」。
  - 案件章里机制符号若属**破案证据本体**（如某线=灵脉被截、某红线=下毒连线），可保留必要揭示，但**不要用「灰痕一跳/某红线连着」这类「个人反应 + fidget」反复贴标签**；证据靠「他凑近看 / 他盯着 / 对方脸色变了」呈现。
  - **不要硬加「被指引 / 被召唤」的被动神秘牵引（属 #012 事实诚实特例）**：主角走向某处、探查某物，通常有明确目的或主动反应，**不必写成「冥冥之中被什么牵引 / 指引」**。"寒意引着他""像被深处牵着"这类被动语态既混淆感知归因（见上），本身也是读者腻味的套路，应删。若需写身体对灵物 / 传承的感应，落在实际感知（传承发烫、识海异样）+ 主角**主动判断 / 目的**（"得去看看""总要到头看个明白"），不写"被X牵 / 引 / 指引"。错误：「越往里寒意越分明，像有什么引着他」（冷当引，归因错）；也错：「阴冷越重，体内传承发烫，像被深处牵着」（归因虽对，但"被牵"仍是被动神秘主义，依旧读者判假）；正确：「越往里阴冷越重，体内传承却渐渐热了起来。他没停步，既进了这石室，总得到最里头看个明白」（感知 + 主动目的，无被指引）。
  - 术语（如查线誓 / 看线 / 线亮）是机制名，可保留；叙事描写层一律不贴标签。
  - 与 #006 关系：#006 治「调用次数多」，本例治「即便次数少了仍直白」——两者叠加才是「读者不烦」的完整解。

## 案例 #008 — 所有角色共享旁白语感（对话措辞无个体差异）

- **来源**：用户短篇项目反馈（科幻题材，2026-07-12）
- **现象（用户原话）**：「感觉角色说的话，措辞语序之类的习惯很像旁白，没有一个人的感觉」
- **根因**：技能有情绪词库（控制**叙述**用词），有对话标签库（控制**怎么说**），但**没有控制「不同角色说话措辞不同」的机制**。导致工程师和伦理顾问、丧偶男人和丧女母亲，对话全部共享同一个叙事语感——句子长度趋同、书面化程度趋同、连转折节奏都一样。
- **错误写法（片段）**：
  > 「不是假。」角色甲看着他，「那声妈，那抹笑，真实发生过——只是别人家，别人的女儿。它对我有没有用，和它属不属于我，是两件事。某 AI 的底子是 coherence over truth，先让你拼成一个能站的自己。我选了拼上。」
  > （角色甲的措辞和角色乙的旁白无法区分——「coherence over truth」「属不属于我是两件事」都是论文/旁白语，不是人说话）
- **正确写法（片段）**：
  > 角色甲摇头。「不是假。那声妈——真的。别人家，别人的女儿。但真的。」她顿了顿。「对我有用，跟属不属于我，两件事。那 AI 不管真假，只管你能不能站。我选了能站。」
  > （拆碎长句、去术语、加口语停顿——角色甲在说话，不是旁白在替她组织措辞）
- **可固化规则**：
  - 技能新增「声口特征」节（角色对话个性化模板，含基础声口/情绪→台词变化/声口对比/盲测），强制每个角色的对话拥有独立措辞风格
  - 写对话前先查声口特征；写完后做声口盲测（拿掉角色名能认出谁在说话）

## 案例 #009 — 角色分析完补"总结尾巴"显得像反派/教唆

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-13）
- **现象（用户原话）**：①「你来找我，是跳出这死结：让上头，替他再断一回。」——"这主角看起来像个反派"；②改成"你来找我，无非是想有人，替你把实话摊到上头跟前——它自会断。"后——"多说多错，你非要后面加一句，就跟像是在教唆"
- **根因**：角色做完推理/点破后，硬补一句「所以你该 X / 你来找我是 Y」的兜底结论，是双坑叠加——①「跳出死结 / 替 X 断」是操纵规则的油滑钻空子腔（像反派）；②居高临下替对方下结论、点明"你该来找我"，像在教唆/说教（也犯 #006/#007 的"太直白上课"）。本质是作者不放心读者自己接，非要角色把潜台词说尽。
- **错误写法（片段）**：
  > 他顿了顿，"他神智昏着，旁人趁虚进谗，断出来的便不公。你来找我，是跳出这死结：让上头，替他再断一回。"
- **正确写法（片段）**：
  > 他顿了顿，"他神智昏着，旁人趁虚进谗，断出来的便不公。"
  > （分析到位即止，不补结论尾巴；立场由后续动作/接案表达，如"我接"）
- **可固化规则**：
  - 角色做完推理、点破、教做人之后，**删除所有「所以你 / 你来找我是为了 / 无非是 / 说到底你就是」类兜底总结句**——让对方或读者自己接。
  - 若需表明主角立场（接案/助人），用**动作或直接许诺**（"我接"）而非嘴上总结"你该来找我"。
  - 与 #008 关系：#008 治"所有角色说话一个调"，本例治"同一个角色说话带教唆/反派腔的尾巴"——声口之外还要看台词的**立场姿态**是否越界。

## 案例 #010 — 台词未查语用限制（人称/谦敬绑定、语体、视角）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-13）
- **现象**：把描写/成语直接搬进角色台词，忽略词语的**使用条件**。例：称某角色台词「化用刻骨铭心」，但「刻骨铭心」是带语体（庄重书面）+ 视角（事后回顾）限制的描述词，让角色**当面、即时**对师长说会立刻"隔"；又如谦敬辞「犬子」只能称己方孩子，绝不可称对方孩子——人称绑定是绝对限制，违反即笑话。
- **根因**：只做修辞归类（像不像、美不美），漏了**语用层**（能不能、该谁说）。修辞管表达效果，语用管使用条件；写台词先过语用，再过修辞。
- **错误写法（片段）**：
  > "您教我的问心二字，我刻在骨头上。"（把"刻骨铭心"拆成生造口语当面说，既不满足"化用需有可辨识源文本"的前提，又撞上原成语的语体/视角限制）
- **正确写法（片段）**：
  > "您教我的问心，我从未敢忘。"（第一人称、当面、即时、对师长——无任何语用冲突，自然语言）
- **可固化规则**：
  - 写台词下笔前查三道语用限制：①**人称/谦敬绑定**（犬子/令郎/家父/尊夫人等绝对限制，谁说、对谁说焊死在词里）②**语体绑定**（书面庄重 vs 口语随意，场合错配即隔）③**视角/时态绑定**（回顾性、旁观性描述词不宜当下当面说）。
  - 修辞是加分项，语用是及格线；语用不过关，修辞再漂亮也不能用。

## 案例 #011 — 把生造表达包装成"化用/修辞"给合法化

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-13）
- **现象**：作者（或分析者）把一个**生造的具象表达**当成"化用成语/修辞妙笔"来正当化，掩盖它其实不合规范。如把「刻在骨头上」讲成"化用刻骨铭心的具象化改写"——既错用"化用"术语（无读者可辨识的源文本），又给生造镀了金边。
- **根因**：混淆"生造"与"合理新意象"；习惯给不规范表达贴修辞术语使其显得高级。
- **错误写法（认知）**：
  > 称「刻在骨头上」为"化用刻骨铭心、具象化改写成语"——错：化用要求源文本可辨识，拆成语当面说不满足；它只是个生造的直白夸张。
- **正确写法（认知）**：
  > 「刻在骨头上」是建在「刻骨」语素上的直白具象夸张/比喻，谈不上高级技巧；它是生造表达，只因生造得合乎汉语直觉、读者秒懂才勉强可用，绝非"化用/妙笔"。
- **可固化规则**：
  - **"化用"有硬边界**：必须存在读者可辨识的源文本，改造在合理范围。把成语拆开随便摆、读者认不出母本，不是化用。
  - **生造分好坏，不镀金**：坏生造=破坏汉语直觉、谁都卡壳的词组（如"教你说见""更慢的东西"）应删；可接受具象新意象=建在已有语素/意象逻辑上、读者秒懂（如顺"刻骨"逻辑展开）。关键是**别把生造包装成化用/修辞妙笔**——生造须受"读者秒懂 + 合乎汉语意象"约束，做不到即病句删。
  - 与 #010 关系：#010 治"用词撞语用限制"，本例治"为生造表达乱贴修辞术语"——两者都要求写手对词语保持诚实，不靠术语给不规范表达开脱。

## 案例 #012 — 用词与场景的"事实诚实"五类坑（脱离事实夸张 / 坏生造词 / 逻辑·场景·指代·重复错 / 错扣 canon 专名 / 机制隐喻硬收）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-13 多轮通读）
- **现象**：五类问题独立但同根——写手对"词语 / 场景 / 设定"不够诚实，靠夸张、生造、错位、机制裸奔来盖过事实或硬造钩子。
- **根因**：下笔前未核对"剧情给定事实基底"与"词语使用条件"，把"读着还行"当成"对"。
- **五子类（错误 → 正确）**：

  **① 脱离事实的夸张**
  - 错误：「一个老人对自己昏聩半生的愧」——角色实则是**近期**被下毒才神识昏聩，并非半生。夸张超出剧情事实基底，读者判假。
  - 正确：「一个老人，对自己昏聵、错怪了人的愧」——落到具体事实。
  - 规则：**夸张须有事实地基**；一旦脱离剧情给定的时长/程度/因果，读者立刻觉得假。

  **② 坏生造词（"似词非词"）**
  - 错误：「露出底下枯、无用的本记」「已涉故意慢害师长」——"本记""慢害"不是汉语词，是拼错的似词非词，读者秒噎。
  - 正确：「露出底下枯、无用的本相」「已涉故意谋害师长」。
  - 规则：坏生造=破坏汉语直觉、谁都卡壳的词组（本记/慢害/教你说见/更慢的东西）应删；可接受具象新意象须合汉语逻辑、读者秒懂（见 #011）。

  **③ 逻辑·场景·指代·重复错误**
  - 错误：「某角色这眼，和那是同一个。」（两人不能共有同一眼睛/眼神；"这眼"指代不清）；「他转身出堂……陈言没打扰，只把这一幕收进心底」（人已出堂又"在堂内看"）；室内写「老泪混着雪沫」（屋里无雪）；同一场和解写两遍（先写认错、后文又写"扶起"，与"已起身"矛盾）。
  - 正确：「某角色这眼神，和那是同一种。」；删提前的出堂+雪景，收束留在堂内；重复和解只留一次。
  - 规则：整句/段过"逻辑自检"——两人能共享此器官/神情吗？场景（室内外、在场离场）与动作自洽吗？同一事件写了两遍吗？

  **④ 错扣 canon 专名 / 错引台词**
  - 错误：本章某案是师徒私事（与某宗门无关），却写「某宗门因他立的誓免了一场死伤」，又引了属别章的台词「照看某宗门」——案件归属错、时间线错、无中生有。
  - 正确：本章只收尾本案件的真相亮明；该宗门/该台词移交其归属章节，不在此硬塞。
  - 规则：专名/台词须严格按 canon 的"所属案件/时间线/人物关系"使用；跨章借用须核对归属，不把别章钩子焊进本章。

  **⑤ 超自然机制隐喻硬收（无铺垫强行钩子）**
  - 错误：章尾「某域水面下那些缠着的因果，今日又分明了一分，像是被人从水底，又拨亮了一缕。」——诗化外衣裸奔核心机制（视觉化符号），两个"又"悬空前文无承接，"分明/拨亮"语义重复；与 #007 不裸奔机制冲突。
  - 正确：删机制隐喻，改为基于本案件事实的角色经验判断——「这一案，某人昏聩、某人下毒、某宗门上下噤声，分明是有人生怕真相见光。主角走南闯北这些年，见过太多这样的遮掩，知道幕后的手脚，往往比台面上的礼数更阴。」
  - 规则：钩子基于剧情事实 + 角色经验/判断来埋，不靠超自然机制的隐喻（水底因果/拨亮一缕/网清晰）强行收束；埋钩子 ≠ 裸奔机制。

- **可固化规则（总）**：写词句/场景前三问——①这词有人称/语体/事实限制吗？（#010/#011）②脱离剧情给定事实了吗？（夸张/生造/错扣）③场景动作逻辑自洽吗、机制裸奔了吗？（#007）诚实面对"剧情事实基底"，不靠夸张/生造/错位/机制隐喻盖过或制造。

## 案例 #013 — 前世/穿越设定正当用法约束（仅限独有资产，不拿来当万能解释）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-14）
- **现象**：穿越/重生文里，主角频繁用「前世见过/跑过/领教过」来证明自己镇定、见多识广、有办法。用户原话：「再往后修为高了都几百岁的老怪物了，你上辈子那三四十年有什么用」「你写的这些都是没事就提两句，还都是没太大关联的点，最多就是体现一下主角见多识广，上辈子见过，但是这是个修仙世界啊」。
- **根因**：把「穿越身份」当万能解释器——任何需要主角显得靠谱处都塞一句「前世 X 过」。但修仙世界里几百岁老怪比比皆是，主角前世三四十年凡人经验毫无优势；且这类引用多与当前情节弱相关，纯属凑设定。
- **错误写法（片段）**：
  > 主角每一步都踩得很稳——他前世跑过不少这样的场子，山要塌、人要疯的前一刻，空气都是这个味道。（用"前世跑过场子"解释镇定，修仙世界观下逻辑崩，且弱相关）
- **正确写法（片段）**：
  > 主角却走得稳当。山路滑，他脚下自有分寸，满山震动未歇，他也不慌。（用本世观察/动作自证，不点穿越）
  > ——真正合理的「前世」用法应仅限：引用来自前世的**独有资产**（法律知识、现代法理思维、律政专业经验）且直接服务当前情节，如「他前世在法庭上最擅长的便是定程序：程序公了，结果再差也服众」。
- **可固化规则**：
  - **前世仅限"独有资产"引用**：只有"前世带来的、修真界没有的专业能力"（法律/法理/现代制度思维）且**直接服务当前情节**（用法律思维解修真困局）时，引用前世才合理，必留。
  - **前世≠万能解释**：用「前世见过/跑过/领教过/当事人的眼神」证明主角镇定、见多识广、有办法的，在修仙世界观下逻辑崩（老怪面前三四十年经验无优势），且多弱相关——**按用途全清，不按频率砍**。
  - 改法：弱相关前世句改为本世自证（动作、观察、经验判断），或删。

## 案例 #014 — 避免过量内心独白/心理推演（"小说要故事，不是主角怎么想"）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-15）
- **现象**：叙述靠主角的"想"推进。用户原话：「太多的心理活动也不对，小说要的是故事，不是主角怎么想怎么想之类的」。典型症状：用「在心里分作两处：一桩已了，一桩才开头」「把那一幕在心里反复嚼……（长段推理）……这东域的因果，往后每亮一分，对面的改，便多一分忌惮」「心头微动，他想……忽觉肩上沉了些，他不知这担子要扛多久，只知道……」等长段内心分梳/推演当叙述发动机。读者读到的是主角的大脑运转，不是事件在发生。
- **根因**：把"交代主角思路"误当成"推进故事"。小说靠**故事**（动作、对话、事态、他人反应）往前走，内心活动只是其中一味调料，不是主菜；一旦用"主角想 X、想 Y、得出结论 Z"替代"事发生"，故事就停了。
- **错误写法（片段）**：
  > 他把那桩刚在寒月宗了却的冤案，和这桩刚接下的麻烦，在心里分作两处：一桩已了，一桩才开头。（纯内心分梳，零情节推进）
  > 他放慢了脚步，把今日誓石认他传承的那一幕，在心里反复嚼。初代……（长段设定推理）……这东域的因果，往后每亮一分，对面的改，便多一分忌惮。（用内心独白倾倒设定伏笔）
- **正确写法（片段）**：
  > 他脚下不停，踩着雪往山上去了。（删内心分梳，动作直接推进）
  > 他回头望了眼洞口——那镇器认了他传承的一幕，还印在眼底。这一程，总算没白来。（长段推理 → 事实 + 一句收束）
  > 设定伏笔交由后续章节的情节/对话自然带出，不在本章用内心一次性倒尽。
- **可固化规则**：
  - **写完一段自问**："这处是靠'他想'推进的，还是靠'事发生'推进的？"若靠想，改事。
  - **内心活动只保留两类**：①情节必需的决策瞬间（"他当即改了主意"）②角色声口下的简短反应（"他心头一紧""他眉头一皱"）。其余长段心理分梳/推演**删**，换成：动作、对话、他人反应、可见事态。
  - 设定伏笔/世界观信息**不靠主角内心推演倾倒**，交给后续情节与对话自然呈现。
  - 与 #007 动作法同源：信息不靠内心标签/机制标签承载，靠可观察的故事；与 #013 互补：穿越设定也不靠"前世想过"补逻辑。

## 案例 #015 — 角色动机须与其实际能力/处境相称（不自相矛盾）

- **来源**：某章写角色前往某处，把行动动机写成"他只是惦着，山那头有一群弱小的人，正被一群怀恨的人盯上"——即"出于善良，要去护住弱者"。
- **现象（用户原话，抽象后）**："这角色自己并没有多少实力，就算有某种特殊能力能在明面上无视些差距，也不能真的把自己当成可以保护其他人的人啊。别人可能被他的表现迷惑，以为他很强，但他自己得知道自己的斤两——所以'当好人发善心'这种理由绝对不成立。"
- **根因**：动机与角色的实际能力/处境自相矛盾。作者把一个**只有"更强者"才成立的目标**（保护/拯救他人、英雄救世）安在了能力并不及此的角色身上，等于叙事替角色自认了"我能做到"，与设定冲突，读者会判假。
- **错误写法（片段 ≤150 字）**：
  > 他不为那机缘来。那句托付他始终没敢忘——他只是惦着，山那头有一群弱者，正被一群怀恨的人盯上。（"惦着弱小被盯上"=以护弱为动机，但角色并无相应实力，无效）
- **正确写法（片段 ≤150 字）**：
  > 他不为那机缘来。那句托付他应下了，便不能当没听见。（动机=承诺/债务，已是充分动机；角色的分寸与手段，交后续情节中的行为呈现，不另写一段说明）
- **可固化规则**：
  - **动机能力相称原则**：角色的行动动机，不能要求他具备自己实际没有的能力/地位。动笔前自问："这角色凭什么相信自己能干成这件事？"若答案是"因为他想帮人/因为他是好人/因为他该出手"，而非"因为他有具体的手段"，就是动机造假。
  - **角色定位须前后一致**：角色的自我认知、旁白对其的定位，必须贴合其真实能力（金手指只在生效领域有用，越界照样无力）。这是*原则*，不是*要写的段落*——靠后续行为呈现，不在动机处另写"他自知斤两"的说明。
  - **诚实动机的三类来源**：①**承诺/债务**（"这话我应下了"）②**牵连自身的利害关系**③**能力边界内的目标**（用自己真能做到的事去影响局面，而非幻想自己能替人挡刀）。
  - **叙事不替角色自我认知**：别人可被角色表现迷惑、高估他，但主角内心/旁白对自己的定位，必须贴合其真实能力——不能借"发善心"偷偷把角色抬成他不是的强者。
  - **诚实动机 ≠ 写一段说明文字**：动机合理，不代表要另起一段告诉读者"他自知斤两/他唯一能倚仗的是某本事"。承诺/债务本身就构成充分动机；角色的能力边界与做法，留给场景中的行动去呈现（与 #014 同源：少告诉、多呈现）。
  - 与 #013（设定资产不滥用）、#014（少写内心）互补：动机的诚实，最终也落在"角色到底凭什么行动"这一事实基底上。

## 案例 #016 — 脱节的信息交代 / 无营养旁白（补内容填感官细节，不填剧情交代）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-15，ch7 通读）
- **现象（用户原话）**：「这种文字出现在这里是为了什么? 凑字数也不是这么凑的啊, 写点环境啊, 长相啊, 衣着啊, 特效啊, 什么东西不能凑? 非要写这种前后毫无关系还毫无营养的东西」
- **根因**：需要补字数/交代信息时，作者用**与当前场景脱节、无感官信息**的旁白把剧情/前因硬塞（如角色已在山上，却写「某管事早先一步把话递到某镇」）。这类交代既不 show（无环境/人物可感），又与当下场景「前后毫无关系」，读者读来是空转补字。
- **错误写法（片段 ≤150 字）**：
  > 他不为那机缘来。某前辈托他照看某门，这话他应下了。某势力要在秘境里跟某门算总账——这话，某商会管事早先一步递到了某镇。（角色在山上，却交代镇上旁人递话；纯信息，零画面）
- **正确写法（片段 ≤150 字）**：
  > 他不为那机缘来。某前辈托他照看某门，这话他应下了。山风顺着领口灌进来，激得他肩背一紧，脚下却更稳了，踩着雪，一步不停地往上。（同处场景内，用环境+动作承载，把「决心」交身体反应呈现）
- **可固化规则**：
  - **补内容填感官，不填交代**：字数不够或需交代信息时，优先写与当前场景相连的**环境 / 长相 / 衣着 / 具体动作 / 特效**，绝不用「谁早先把什么话递到哪」这类脱节的剧情信息硬塞。
  - **旁白须贴着当下场景**：凡写「某人事先得知 / 某处早有风声」，先问「这与角色此刻所处场景有关吗？」无关即删，或改为场景内可感之物（风、雪、路、人影）。
  - 与 #012⑤（机制隐喻硬收）、#014（少内心）、#015（诚实动机≠写说明）同源：皆「作者跳出来替剧情/角色做交代」的反面；本例特指**脱节且无感官的补字交代**。

## 案例 #017 — 叙事层内心总结/作者点题感慨裸奔（同一件事动作+点评写两遍）

- **来源**：用户长篇连载项目反馈（修真题材，2026-07-16）
- **现象**：一段里先写角色用动作完成某事（捡了某物/做了某取舍），紧接一句「这下 X 够了」「小人物的 X，够用便好」「这点东西上不得台面，却是一份实打实的进项」之类的内心总结/作者点题感慨。读者读到作者借叙事口吻替角色把潜台词喊出来，且同一信息动作里已呈现一遍、感慨里再来一遍 = 双重啰嗦、出戏。
- **根因**：作者不放心读者自己接，非要借叙事层补一句点题。但小说靠呈现不靠点评；动作已承载意思，感慨是冗余复述。
- **错误写法（片段 ≤150 字）**：
  > 他指尖一掐，连根带土塞进怀里，又顺手捞了块碎石料，在掌心掂了掂。分一点机缘，够了。他将草与石贴身收好，又借乱瞥了眼石窟——灵光更盛。他没凑那热闹，小人物的机缘，够用便好。指尖还沾着草根的湿泥，他拢进袖里——这点东西上不得台面，却是一份实打实的进项。
- **正确写法（片段 ≤150 字）**：
  > 他指尖一掐，连根带土塞进怀里，又顺手捞了块碎石料，在掌心掂了掂，草石收好，指尖还沾着草根的湿泥，随手拢进袖里。又借乱瞥了眼石窟——灵光更盛，几道身影正往里挤，他没凑那热闹。
- **可固化规则**：
  - **叙事层不裸奔内心总结/作者点题**：角色做完动作/选择后，删紧随的「这下够了/够用便好/实打实进项/说到底就是 X」类感慨句。意思已由动作承载，感慨是冗余+出戏。
  - **同一件事不写两遍**：动作流呈现过的（捡了草石=分了机缘），不要再发内心总结复述；呈现与点评分居，不要两遍。
  - 与 #009（角色台词补总结尾巴）同根——#009 治台词里，本例治叙事层；与 #014（少内心独白）互补——#014 治长段心理推演当发动机，本例治短促感慨/点题句；与 #016（脱节信息交代）同源但本例特指贴着场景却画蛇添足发感慨复述。

## 案例 #018 — 做局情节的「零增量催化」与「外部规则降智解」

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-16）
- **现象（抽象）**：冲突段的「发动机」有两个常见烂法——①某角色 A 一直盯着目标 X，作者却让配角 B「指给 A 看 X」，等于让一直盯 X 的人去「看 X」，动作对目标和机制都无增量；②反派作恶时，某世界观机制（如某圣物/天道）自动跳出来惩罚反派，主角毫无作为，反派像被设定而非被主角「做」掉。
- **根因**：①把「做局」写成机械降神式的催化——角色动作不带来任何新信息/局势变化，只负责「触发下一幕」；②弱主角文里，作者嫌主角能力不够，偷偷用外部规则替主角收拾反派，主角能动性被架空，与「弱主角靠脑子」定位冲突。
- **错误写法（片段 ≤150 字）**：
  > 配角抬手指向那少年：「看，就是他。」可那反派从头到尾都死盯着那少年，这一指毫无新意。下一刻，某圣物骤然亮起，自动将反派定在原地——主角站在一旁，什么也没做。（「指」零增量 + 机制自动罚反派，主角空心）
- **正确写法（片段 ≤150 字）**：
  > 主角把满场的恩怨线扫了一遍，最绷的是西边两拨人的死仇线。他踢块碎石种下火，那线崩断，混战漫过反派侧翼——反派阵脚一乱，主角趁空档把目标拉出包围。（主角靠「读线+扯线+借乱」制造真实局势变化，反派是被主角做掉的）
- **可固化规则**：
  - **做局动作须有真实增量**：每个布局动作必须给目标/机制带来角色原本不知道、或局势原本没有的新东西（新信息、新位置、新冲突引爆点）。禁止零增量催化：让一直盯 X 的人「去看 X」、当众「揭穿」早已公开的事实、纯仪式性「指一下」。
  - **避免外部规则降智解**：弱主角文的冲突，优先靠主角能力（洞察/布局/借势/趁乱）制造真实局势变化来解决；某机制若作为情节发动机，须让主角动作对其产生真实触发增量——否则它更适合做「威慑/伏笔」，而非高潮解法。反派须被主角「做」掉，不被设定自动收拾。
  - 与 #015（动机能力相称）、#014（少内心）互补：主角的「做局」本身就是其能力呈现，不需要外部机制代劳。

## 案例 #019 — 同一视觉意象/设定词勿在相邻段落反复夹写（集中到关键处单独成段）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-16，某章开头通读）
- **现象（抽象）**：某核心设定视觉化符号（如角色特有的「感知线/因果线」类意象）在开头数段被反复提及——几乎每段都夹一句（「画面里人人都牵着线」「角色问：他们也有线？」「那缕线」「碑身缠着无数线」「这些线不是死的」「只看那些线便知……」），读者被同一意象刷屏，且每段往往只"提一句"而无新信息，读感烦。
- **根因**：作者担心读者记不住设定，于每个场景切换处都补一句意象提示；但设定视觉化只需在「第一次出现（画面）」和「情节关键处理处（集中解读）」两处落地，中间的过渡段不必每段重申。
- **错误写法（片段 ≤150 字）**：
  > 壁上人人腕上都牵着一根线。林小月问：「他们腕上也有线？」他答：「人人都有。」……像有根线在底下拽着他走。越往下那缕线越清晰，竟分出几股。……碑身缠着无数线。……这些线不是死的。……他只看那些线便知：千万缕缠成网……
- **正确写法（片段 ≤150 字）**：
  > 壁上人人腕上都牵着一根线，连向那截碑（首次画面，一笔带过）。……（下行过渡段不提线，只写体感/环境）……他眯眼，果见一缕极淡的冷线从深处漫出（异域线钩子，单独一段）。……（进石室仅画面一笔「碑身缠着无数线」）。……主角定睛去看那些线。它们不是死的：有的绷着有的缠错……织成一张网（集中大段，唯一的解读性描写）。
- **可固化规则**：
  - **设定意象两处落地即可**：①首次出现——用画面/动作一笔建立（读者看见即可）；②情节关键处理处——集中成一段完整描写（解读/作用/变化）。两者之间（过渡、环境、对话）不每段夹写该意象。
  - **同一意象勿刷屏**：若某章某一符号在连续多段都出现，检查是否每段都只"提一句"却无新信息——是则合并到关键段，过渡段用其他可感细节（环境/体感/动作）填充。
  - 与 #007（动作法，不裸奔机制）+ #016（脱节旁白，补感官不补交代）一致：意象靠场景画面与关键动作承载，不靠每段提醒式旁白。

## 案例 #020 — 套路化句构「X了一分」与「人物台词塞旁白描写」

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-16，ch8 通读 + 跨章扫描）
- **现象（抽象）**：两类偷懒写法反复出现——
  ① **固定量化句构「X了一分 / X了几分」**跨章反复当"心理/氛围推进器"用（「心里那点疑，又沉了一分」「那根绷着的弦，又紧了一分」「把他心底那口井的轮廓，又描清了一分」），读者一眼认出是作者的习惯动作，且每章都来一下，读感腻。
  ② **人物台词里塞 narrator 的视觉/比喻描写**：让角色用 narrator 的紫腔形容自己身上的东西（「腕子上那旧纹……它自己热起来了，金闪闪的，像要浮出来」）。真实的人不会用「金闪闪的，像要浮出来」描述自己胎记——这是把本该由 narrator 呈现的感官细节，借角色之口说出，既假又越界。
- **根因**：①图省事，用固定增量句推进心理/氛围，缺变化；②把"呈现感官"误当成"角色能说的话"，角色台词边界失守。
- **错误写法（片段 ≤150 字）**：
  > 他记起外头誓石也认过她，心里那点疑，又沉了一分。
  > 她点头：「腕子上那旧纹……它自己热起来了，金闪闪的，像要浮出来。」
- **正确写法（片段 ≤150 字）**：
  > 他记起外头誓石也认过她，垂下眼。这丫头的来历，怕不简单。
  > 她点头：「那旧纹……自己热起来了，烫得慌，一跳一跳的。」（人物只说真实体感，视觉/比喻留给 narrator）
- **可固化规则**：
  - **忌固定量化句构**：心理/氛围推进禁用「X了一分 / X了几分」套构当习惯动作；每章至多一次，且优先用**具体动作/状态描写**承载（「垂下眼」「绷到了极处」「描清了轮廓」→ 改「把那口井的轮廓，又看清了一层」或干脆用可见事态）。自检时全局搜「了一分 / 了几分」，超 1 处必改。
  - **人物台词只说角色当场能说出口的实话**：体感（烫/痛/慌）、情绪、信息。不得替 narrator 做描写（颜色词、比喻、外形评价）。视觉/比喻/外形交给 narrator 在台词之外呈现。
  - 与 #006（复读）同根——#006 治词面重复，本例治**句构套路化**与**台词越界**两类；与 #007（动作法，不裸奔机制）、#017（叙事层总结）互补：都是"作者偷懒用固定套路替代具体呈现"。

## 案例 #021 — 「拿着答案问问题」(loaded question) 与 非亲密关系写贴身站位

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-16，ch8 四修）
- **现象（抽象）**：两类"根子没动"的假修改——
  ① **拿着答案问问题**：角色 A 已经看见/知道某事实（对方捂着受伤处、前文明明交代过对方有此反应），却张口问一个预设了答案的问题（「手腕又不对劲？」「你又生气了？」），等于作者借角色之口把已知信息硬递给读者。读者一眼看穿"这问题作者早知道答案"，对话变刻意、变 lecturing。
  ② **非亲密关系写贴身站位**：两个关系分明是师徒/同伴/上下级，作者却让一方"贴着"另一方、往对方肩/怀里缩——哪怕删了"呼吸喷在手上"这类露骨句，身体紧贴的站位本身已暗示暧昧/恋慕，与关系设定冲突。
- **根因**：①把"对话"当"信息传递工具"——作者想确保读者收到某信息，于是让角色问一个已知道答案的问题来"明示"。但好对话由**可见行为**触发，角色问的是他当下真不知道、且能由对方行为合理引出的事。②把"亲密站位"当"保暖/依赖"的默认写法，没意识到站位本身传递关系信号；关系非亲密时，物理接触须落在衣服/身后，不得落在身体紧贴。
- **错误写法（片段 ≤150 字）**：
  > 她捂住左腕。他低头：「腕子又不对劲？」（他早看见她捂腕、前文也知她腕子会热，这问题零信息量，纯递梗）
  > 冷窟子里，她贴着他，冻得直往他肩里缩。（师徒关系却身体紧贴，删呼吸仍暧昧）
- **正确写法（片段 ≤150 字）**：
  > 她捂住左腕，呀了一声。他低头：「怎么了？」（genuine 开放问，由她行为引出；腕子信息由她自答自然长出）
  > 她跟在他身后，攥着他衣角，冻得缩了缩脖子。（接触衣服非身体、缩脖子是怕冷本能，自然拉开距离，师徒站位成立）
- **可固化规则**：
  - **对话须由角色真不知道的事触发**：角色提问，应是他**当下无法从可见行为推出**的疑问（「怎么了？」「手怎么了？」），而非预设答案的 loaded question。已知信息让对方用**行为/自答**自然带出，不靠提问喂。自检：凡角色问句含「又/果然/早就」类预设词、且答案作者心里门清——必改 genuine 问法。
  - **物理站位须匹配关系**：非亲密关系（师徒/同伴/上下级）禁用身体紧贴站位（贴着/往怀里缩/肩并肩蹭）；保暖/依赖用「跟在身后 + 攥衣角/袖口 + 缩脖子/搓手」等落在衣服或身后、且本能拉开距离的动作承载。恋慕/暧昧站位仅留给确属亲密关系的角色。
  - 与 #008（声口差异）、#020（台词只说角色能说的话）互补：本例治"对话的诚实性"与"站位的关系信号"，都是让文字不越角色本分。

## 案例 #022 — 超自然效果的「巧合式一闪」与 阴森/氛围段堆虚比喻

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-16，ch8 五修）
- **现象（抽象）**：两类"作者痕迹"重的写法——
  ① **超自然视觉效果写成"一闪即灭恰被主角逮到"的巧合**：某异象（金光/纹路/虚影）「浮起一层极淡的金，转瞬又灭，快得几乎像是错觉」，偏偏正好被主角这一瞬看见。读者会觉得"这么短的效果凭什么正好被逮到"，是作者硬安排的巧合。
  ② **阴森/幽暗/静谧的环境氛围段堆"虚比喻"**：写地下石室、密林、古墓这类氛围，连用「连呼吸都像凝住了」「静得反常」「仿佛踏入另个天地」等"像/仿佛+抽象喻体"的虚描写。喻体空泛、彼此同质，读来不但不吓人，反而假、腻、出戏。
- **根因**：①作者想"秀"一下神秘效果又怕过度暴露设定，于是让它"一闪即灭"——却制造了巧合；②以为氛围靠"比喻叠加"来渲染，实则抽象喻体越多越假，真正的氛围来自具体感官（光/声/温/湿/触）。
- **错误写法（片段 ≤150 字）**：
  > 那道古纹浮起一层极淡的金，转瞬又灭，快得几乎像是错觉。（巧合式一闪）
  > 四壁幽幽地照，连呼吸都像凝住了。满室静得反常，仿佛踏入另个天地。（氛围段堆虚比喻）
- **正确写法（片段 ≤150 字）**：
  > 她摊开手给他看，那道古纹泛着一层极淡的金，随腕子的热一跳一跳。（效果持续/由角色主动展示，主角看见顺理成章）
  > 四壁的光落在人脸上，泛出一层青。头顶那些喊杀声到这儿只剩闷闷一线，隔着厚厚的岩层，断断续续。（用光色+声音衰减落地，不用虚比喻）
- **可固化规则**：
  - **超自然效果避免巧合式一闪**：异象若需被主角观察到，让它**持续一段**、或由**角色主动展示/触碰引发**，而非"闪一下正好被逮到"。"快得像错觉"这类句式是巧合信号，见即改。
  - **氛围段慎用虚比喻**：写阴森/幽暗/压抑/静谧，优先用**具体感官**（光的颜色与强弱、声音的衰减/回响、温度、湿度、气味、触感）落地；少用"像/仿佛/仿若+抽象喻体"（凝住/另个天地/反常）。每个氛围段虚比喻至多一处，且须有具体载体支撑。
  - 与 #007（动作法，不裸奔机制）、#012（事实诚实）一致：氛围与超自然都靠可感的具体呈现，不靠抽象喻体或作者安排的巧合。

## 案例 #023 — "Noun一Verb" 四字结构（重复率高的廉价叙述节拍：含体内 tell 与体外事件 cliché）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-16，ch8 六修 + 七修）
- **现象（用户原话，两批合并）**：
  ①「猛地一热 虽然不是x了一分，但是感觉是一样的，这种写法你还有很多，我觉得你应该用一来判断是否出现这种写法」——体内突发体感 tell。
  ②「我不同意你要保留的碑身一震/虚影一颤/台心一颤这些所谓的外部事件或动作，你完全可以换别的词语，没有四个字的就用一句话描写，非要写这种重复率极高的东西出来吗？」——即便是"外部事件"，凡"名词一动词"四字模板（碑身一震、虚影一颤、台心一颤、脚步一顿、脚下一顿、脸一沉、一字一顿、光影一晃、齐齐一颤）也属重复率极高的偷懒写法，不能保留。
- **根因**：作者用一句固定四字模板替代具体描写——要么把情绪外化成"体内猛地一X"的短 tell（常伴随来路矛盾），要么遇到任何"动作/事件停顿"就套"Noun一Verb"（碑身一震、脚步一顿、脸一沉），全书的转折与反应全靠这几副面孔，重复率极高、毫无新意。
- **错误写法（片段 ≤150 字）**：
  > 碑身一震，体内那点传承猛地一热，一股暖流顺着小臂涌上来，像游子归了巢。（体内→小臂来路矛盾 + 猛地一热 tell + 碑身一震 事件 cliché）
  > 陈言脚步一顿，余光扫过——（"脚步一顿"在 ch8 同一章出现两次，纯模板复用）
- **正确写法（片段 ≤150 字）**：
  > 石面在他掌下有了回应，竟透出暖意，顺着小臂缓缓漫上心口——像漂泊多年的人，终于又踏进了家门。（用一句话写碑的回应，不套四字）
  > 陈言停住脚，余光扫过——（一处用"停住脚"，同章另一处用"收住脚"，不重复同形）
  > 他正看得出神，碑周的光影无声荡开，像水面被风拂过。（"忽地一晃"→展开成一句描写）
- **可固化规则（检测用「一」+「名词一动词」）**：
  - **判定边界**：凡"名词+一+动词"四字（或"忽地/猛地/齐齐+一+动词"）用作叙述节拍，无论体内体感（心头一震、体内一热、瞳孔一缩、指尖一颤）还是体外事件（碑身一震、虚影一颤、台心一颤、脚步一顿、脚下一顿、脸一沉、一字一顿、光影一晃、齐齐一颤），一律视为重复率高的廉价写法，要清。
  - **因果线/古纹 motif 的窄例外**：本书核心视觉机制"因果线/纹路"允许用颤动、发亮、一明一灭、一跳一跳等描述，但（a）**同一章内不重复同一四字形式**；（b）优先换动词（颤/亮/沉/绷/浮/荡）而非固守"一X"模板；（c）林小月古纹"一跳一跳"可作 motif，但同章第二次出现须换写法（如"微微起伏"）。
  - **替换方向**：要么给明确外部起因/具体描写（用一句话写反应），要么直接写认知或动作；没有现成四字就展开成一句，**绝不硬套"Noun一Verb"**。
  - **写后自检步骤**：grep 全文 `一(震|颤|顿|沉|晃|响|跳|亮|明|灭|缩|荡|醒|紧|暖|麻|凉|热|烫|疼|松|漾|锐|空)` 与 `猛地一|忽地一|陡地一|蓦地一|齐齐一|心头一|体内一|脊背一|肩背一|手心一|瞳孔一|指尖一|眉心一|碑身一|虚影一|台心一|脚步一|脚下一|脸一沉|一字一顿` ，逐条判是否为"名词一动词"廉价节拍（因果线 motif 按上条窄例外放行并检查章内重复）；同时 grep `了一分` 抓 #020 残留。
  - 与 #020 同源：都是"小体感/小事件突变量词"信号；#020 用"分"做程度量词、本例用"一+动词/四字模板"做突发量词。合并视作同一禁令的两副面孔。

## 案例 #024 — 标点符号滥用（破折号/省略号/强调引号堆叠，AI 写作典型痕迹）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-17，ch8 九修：用户原话"ch8还有大量的符号乱用——太多了"）
- **现象（用户原话）**：「ch8还有大量的符号乱用 —— 太多了」——指破折号（——）在 3600 字体量里出现 22 处（约每 150 字一个），另有多余省略号、以及把「」当强调/概念引号反复堆（"对不上"×3、"验"、"问"、"枷锁"）。
- **根因**：作者把破折号当万能连接符——appositive/aside/转折/比喻引子全用「——」；把省略号当犹疑/停顿填充；把「」当 air-quote 强调普通词。三者叠加 = 全文符号密度过高，是 AI 生成文最显眼的特征之一，读者一眼出戏。
- **错误写法（片段 ≤150 字）**：
  > 陈言看得出神——壁上刻的桩桩件件，都是讼师为人立誓、辩冤断案的旧事。（转折/解说硬套破折号）
  > 钱万三那本「对不上」的账册……东家疑那些「对不上」的裁……碑会「问」人。（同一概念词被「」标 3 次 + 省略号滥用）
- **正确写法（片段 ≤150 字）**：
  > 陈言看得出神，壁上刻的桩桩件件，都是讼师为人立誓、辩冤断案的旧事。（逗号/句号替代破折号）
  > 钱万三那本对不上的账册……东家疑那些对不上的裁……碑会问人。（去掉强调引号作普通短语；省略号仅留对话自然断续）
- **可固化规则（写后自检硬性指标）**：
  - **破折号（——）**：单章（3500–4500 字）出现 **≤2–3 处** 为正常；超出一律改逗号或句号。仅两种情形可留：①引号内表示**物理文本被截断**（如碑文刻到一半劈断「天道公正，实为枷锁——」）；②**真正的突然打断/话题硬转**且逗号句号确实不通。凡"解说/举例/比喻引子（像…）/并列说明"前的「——」一律改逗号。
  - **省略号（……）**：仅留**对话中角色自然犹疑/声音断续**、或**神秘声音/传达被掐断**处；叙述性省略号（"他沉默了……""往事如烟……"）一律删，用动作/环境承载。不堆。
  - **强调/概念引号（「」当 air-quote）**：普通词/概念（对不上、验、问、枷锁、锁…）**不用「」标**，作普通行文；「」只用于①人物原话对话；②引述碑文/手书/文献原文；③书名/篇名。同一概念词在章内被「」标 ≥2 次即违规。
  - **写后自检步骤**：grep 全文 `——` 数个数（>3 必改）；grep `……` 逐条判是否在对话/声音；grep `「[^」]{1,4}」` 抽查是否为强调引号（非对话/引文/书名）。
  - 与 #006（符号过度复读）关系：#006 治"同一符号在文中反复复读同一信息"，本例治"符号种类本身滥用/密度过高"，是更底层的标点卫生。

## 案例 #025 — 否定式冗余描写（"不X不Y"打预防针 + 模糊拟人比喻 + "不是"澄清冗余）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-17，ch8 微调：用户原话"不伤人不扑人 像在问他什么 都是老问题了"）
- **现象（用户原话）**：「不伤人不扑人 像在问他什么 都是老问题了」——指 L25「石室四壁浮起几道虚影，不伤人不扑人，只一圈圈绕着陈言转，像在问他什么……这不是攻杀，是传承在验他」中三处同源问题：①"不伤人不扑人"；②"像在问他什么"；③"这不是攻杀"。
- **根因**：①**否定式打预防针**——作者怕读者误会虚影有攻击性，提前用"不X不Y"说它"不做什么"，而非写它正面"在做什么"；读者要的是正面动作，预防针是废话。②**模糊拟人比喻/贴意图标签**——"像在问他什么"给虚影贴了个"在问"的意图，与"像见了自家人""终于踏进家门"同族，用模糊比喻代替具体展示；"在问/在验"的含义应由后文具体事件让读者自己感知。③**否定式澄清冗余**——"这不是攻杀"与"不伤人"重复表达同一意思（虚影无攻击性），前文无攻击暗示时此句纯多余。
- **错误写法（片段 ≤150 字）**：
  > 石室四壁浮起几道虚影，不伤人不扑人，只一圈圈绕着陈言转，像在问他什么。头一道是个后生被指为盗……这不是攻杀，是传承在验他。（否定式冗余 + 模糊拟人 + 否定澄清，三处同源）
- **正确写法（片段 ≤150 字）**：
  > 石室四壁浮起几道虚影，只一圈圈绕着陈言转。头一道是个后生被指为盗……陈言眯眼。这是传承在验他：看不看得清一段缠错的因果。（只写正面"绕着转" + 具体案例展示"验"，不靠否定/比喻打预防针）
- **可固化规则（写后自检排查信号）**：
  - **写"在做什么"，不写"不做什么"**：凡"不X不Y"结构（不伤人不扑人、不攻不守、不悲不喜…）出现在描写里，先判它是不是在"打预防针说某物不做什么"——是则删，直接写正面动作；成语/合理副词状语（不声不响地退下）保留，但"不X不Y+动作对象"式防御性描写一律删。
  - **模糊拟人比喻 = 贴意图标签**：grep `像在[问做看等说]|仿佛在[问做]` 抓"像在问/像在等/像在说"类，改由后文具体展示该意图，不在描写里替物/人下意图结论。与 #007（不贴机制标签）、#022（虚比喻堆叠）同源。
  - **"不是"澄清冗余**：grep `不是[一-龥]{1,4}[，。]` 抓"这不是攻杀/这不是幻觉"式，看前文是否已有正面陈述使该否定多余（无攻击暗示时"不是攻杀"删）。
  - **写后自检步骤**：grep 全文 `不[一-龥]{1,3}不[一-龥]{1,3}`（抓不X不Y，逐条判是否防御性冗余）、`不是[一-龥]{1,4}[，。]`（抓否定澄清）、`像在[问做看等说]`（抓模糊拟人）；三条任一命中即按上列规则清。
  - 与 #007（动作法/不贴标签）、#022（虚比喻）、#024（符号卫生）一致：靠正面具体呈现，不靠否定/比喻/符号打补丁。

## 案例 #026 — 配角要有自己的「主线」（立体 = 各司其人设，不为主角主线服务）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-17，ch8 配角戏份连环诊断后的原则升华）
- **现象（用户原话）**：「要让人物立体，需要的是每个人符合自己的人设，他们要做的是符合他们身份的事，而不是往主线上靠，配角应该有自己的『主线』」。此前两轮反馈是其表征：①「写主角太多了，配角没戏份，会提前让读者疲劳」（配角是主角的跟班/道具，无独立场景）；②「配角知道太多、来秘境寻宝还这么有目标」（配角言行全为给主角递主线谜底，成了信息投递员）。
- **根因**：作者以**主角主线为唯一轴心**，把配角当成挂在这条轴上的功能件——配角出场只为①衬托主角、②给主角递信息/伏笔、③触发主角下一步行动。于是配角说的话、做的事都"往主线上靠"，既导致戏份空心（无独立场景），又导致越界知情（替作者提前把主角该慢慢揭的谜底说出来）。本质是配角没有属于自己的目标与逻辑。
- **错误写法（片段 ≤150 字）**：
  > 某商会管事在秘境深处翻找，见主角来，张口便是：「小人奉命寻某前辈留在此处的手书，东家说这底下或许有能解『那桩对不上的事』的物事。我家东家疑那些错裁是有人动手脚，已查了数十年。」（管事的每句话都在给主角投递主线谜底，他自己没有独立于主线的目标，是个长了嘴的伏笔投递机）
- **正确写法（片段 ≤150 字）**：
  > 某商会管事在秘境深处翻找旧文书契约——东家风闻这里早年有修士驻过，留下的东西能变现，这是他此行的正事。闲谈里他提起自家旧账：家乡一位老讼师当年不明不白没了。他要找的、惦记的都是自己的事；那页写着关键线索的残纸，是他翻旧物时顺带撞见、随手递给主角的。（管事有自己的营生目标+私人旧账，行为先服务自己，与主线的交点自然发生）
- **可固化规则**：
  - **每个配角要有自己的「主线」**：一个独立于主角主线的目标/处境/牵挂（营生、旧仇、护家、升迁、自保、心结…）。配角出场时先问："他此刻在为**他自己**的什么目标做事？"若答案是"为了让主角知道 X / 为了推主角去做 Y"，即不合格，重写。
  - **做符合身份的事，不为主线服务**：配角的言行首先由**他的身份+他的目标**决定（商人算钱、护卫听命、弟子好奇、官吏守规），而非由"主线此刻需要什么"决定。主线信息若要经配角之口带出，须是他为自己目标行动时**自然、顺带**产生的副产品，不是他专程来投递的。
  - **配角不预知主角该慢慢揭的谜底**（承 #配角越界知情）：主线核心悬念的答案，不能提前塞进配角嘴里当台词。配角至多掌握与**他自己主线**相关的碎片，且以他的立场解读（商人只关心"能不能变现/对不对得上账"，不关心"天道收割"这种主角层命题）。
  - **偶尔切到配角视角**（承角色戏份约束）：关键章可短暂切配角视角，写他独自为自己目标做的事（管事私下核账、护卫回宗门的不甘、弟子自己琢磨线索），让读者看见配角"离开主角也在活着"。
  - 与 #002/#004（设定一致）、#015（动机能力相称）、#008（声口差异）互补：#015 治主角动机诚实，本例治**配角有无独立存在逻辑**；三者共同指向"每个角色都按自己的人设与处境行动"，而非都围着主角转。
  - **本案例（2026-07-17 ch8-ch11 复检）提炼的具体「往主线上靠」信号**（写作后逐条自查，命中即改）：
    1. **反应式递谜底**：配角因"看出主角动了容/主角问起"才掏出关键线索/道具，把主线答案递到主角手上（如「管事似看出他动了容，从袖中抽出残纸」）。→ 改：配角本就揣着该物/想办自己的事，与主角调查是**两条线自然交汇**，非服务主角。可机检提示：写配角"递信息/亮道具"前，检查触发词是否含"看出他/见他/听他/应他"——若有，多半是反应式服务，需重写成配角自有需求驱动。
    2. **动机是"成为主角"**：配角（尤其弟子/晚辈）的渴望被写成"学您/像您一样/跟着您去干 X"，即把主角的活儿当自己的目标。→ 改：写配角**自己的底气/自己的处境**（如「日后被人欺了，我也能自己站出来说理」而非「我也能学您去告状」）。
    3. **靠主角当鉴定器/工具人**：配角反复"烦您掌眼/请您定夺/等您来办"，把主角当唯一权威，且同一章内复读该句式。→ 改：配角自己先判断/自己行动，只在确有需要时请主角协助；同章内换说法避复读。
    4. **"替某人记着/为某人做"**：配角的记忆/行动动机被写成"替主角记下了/为先生做"，本质是服务于主角。→ 改：写配角**自己**记挂/关心（如「先生为他打了寒噤，那便也是她该记着的人」——是她自己记挂冤者，不是替主角记）。
    5. **被动受庇护**：弱势配角全程只"望主角/磕头/被救"，无一句为自己辩护或推动自身线索的话。→ 改：补其主动动作（如被带走时哑声报出自己的证人「桥洞——阿福认得我！」），让他为自己抗争、推动自身线索。

## 案例 #027 — 「刻在心里／暗中记下」式默记线索套路（信息处理偷懒句）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-17，ch8 线索处理句诊断）
- **现象（用户原话）**：「只把这几个字刻在了心里。不要总是刻在心里、暗中记下什么的」。主角每次得到线索/听到关键信息，作者就用「刻在心里／记在心里／暗中记下／默默记下／把这笔账记下了」一类句子收尾，通篇高频复用，成了处理线索的万能套路句。
- **根因**：作者用一句固定的"内心归档"动作替代真正的人物反应，回避写"这条线索对他意味着什么、他接下来的态度/取舍是什么"。它既是**复读**（同一句式反复出现），也是**信息处理偷懒**——把"角色如何消化信息"简化成一个仓库入库动作，读者看不到人物的判断、情绪或选择。连带地，配角的记仇/记情也常被写成同一个「记下了」，加剧重复。
- **错误写法（片段 ≤150 字）**：
  > 陈言没接腔，只把这几个字刻在了心里。……（隔几段）这笔账，王虎记下了。……（再隔几段）「陈讼师，这一程，小人记下了。」（三处「记下」高频复现，主角对线索、反派对仇、配角对情分全用同一个入库动作，无一处写出具体态度/情绪）
- **正确写法（片段 ≤150 字）**：
  > 陈言没接腔。那半句残文他看过一遍就够，管事要不要掌眼是另一回事，他没应承。……这笔账，他咽不下去。……「陈讼师，这一程多承照应。」（主角落回"取舍/态度"，反派落回"情绪"，配角落回"谢辞"，各写各的具体反应，不再共用「记下」）
- **可固化规则**：
  - **禁止用「刻在心里／记在心里／暗中记下／默默记下／(把某事)记下了」作为线索/信息的处理收尾**，尤其禁止全文高频复用。得到线索后，写角色对它的**具体态度、判断或取舍**（信不信、要不要用、和已知的哪条对上、下一步做不做），而非"存进心里"这个无信息量动作。
  - **区分保留**：字面动作义的"记下"可留（如"他随手记下的猜想"描述某人写笔记的物理动作、"记账/记名字"等实务），要清的是**用作心理归档套路**的那类。
  - **配角的记仇/记情也别都写「记下了」**：记仇→写情绪（咽不下、忍不下、盯着某人背影）；记情→写谢辞/回礼意向（多承照应、来日必报），避免与主角的线索处理句撞成同一个词。
  - 与 #006（符号/词语过度复读）、#020（套路化句构）同源：本例是"信息处理"场景下的专用套路句。
  - **情报/记账类角色"即时归册"是具象动作，非本套路**：如钱万三商会情报角色当场把线索"归到蓝皮册同页"、账房"入账"——这是该角色自身主线的具体行为，与"主角/配角把线索往心里存"的偷懒句构不同，保留。判别点：是否落在可观察的外部动作（翻册/落笔/归档）而非"心里/心底"内部入库。
  - **写完一章必用全覆盖 grep 自查**（模式不全＝假绿，会漏网，见下"坑"）：`刻在.*心里|记在.*心里|记在心底|压进.*心底|压在.*心底|藏在.*心里|藏在.*心底|收进.*心底|收在.*心底|搁在.*心里|存进.*心里|沉进.*心底|暗记|暗自记|默默记|暗中记|记下了`，命中须逐处判是否"心理归档套路"，>1 处即换写。注意「记在心里」「压进心底」等变体缺一不可，首轮若只 grep 部分变体（如只查「记下了/刻在心里」）会漏掉「记在心里」「压进心底」类写法——这正是实测漏网根因。

## 案例 #028 — 内部属性自相矛盾（同一对象被赋予互斥属性）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-17，ch8 L73 温度矛盾诊断）
- **现象（用户原话）**：「林小月忽然又呀一声，攥紧他：『先生，我腕子又热了，跟这门上的纹，一个冷法儿。』这段看不懂啊，到底是热还是冷」。角色先称某物"热"，紧接着又用"冷法儿"把**同一对象**归为冷，当场自相矛盾，读者无法判断属性。
- **根因**：作者脑中同时装着两层设定（视觉色＝冷金、体感＝热；或冷源↔热反应同源），落笔时挑了矛盾的词直接贴在同一处描写上，没意识到"热了"和"冷法儿"指向同一个腕子、互斥。本质是**一句话内/相邻句内对同一对象的属性断言互相打架**，属于事实一致性的微观漏洞（#012③ 逻辑/场景错的下位）。
- **错误写法（片段 ≤150 字）**：
  > 林小月攥紧他：「先生，我腕子又热了，跟这门上的纹，一个冷法儿。」（腕子刚说"热"，又用"冷法儿"把同一腕子定性为冷——读者当场懵：到底热还是冷？）
- **正确写法（片段 ≤150 字）**：
  > 林小月攥紧他：「先生，我腕子又热了，跟这门上的纹，像是一个来路。」（"热"保留为体感，删掉"冷法儿"这个互斥定性，改用"一个来路"表达两者同源呼应——与后文叙事者确认"门里寒意和她腕上热一般来历"形成呼应，矛盾消解）
- **可固化规则**：
  - **同一对象在同一处/相邻处的属性断言必须自洽**：写完一段涉及"温度/颜色/大小/材质/状态"的描述，回看：是否前句说 A 属性、后句又贴了互斥的 ¬A 属性到**同一个东西**上？若互斥且指同一对象，即矛盾，必改其一。
  - **"冷源↔热反应"类设定的写法**：若设计"冷的东西触发角色发热反应"，描写要分清**两层**——①源的属性（门渗寒意/阴刻冷纹）；②角色体感（腕子发热）。两者可"同源"但**不可把角色的体感直接说成冷的**。表达呼应用"一个来路/同源/应上了"等**关系词**，而非把源的属性词错贴到角色身上。
  - **视觉色 ≠ 体感温度**：若古纹"泛着冷金"（视觉冷色调）但角色感到"热"（体感），这是两层不同维度，落笔时别把"冷金"的"冷"顺手带到体感句里当成温度。
  - 与 #012③（逻辑/场景错）、#002（设定一致性）互补：#002 治跨章设定矛盾，本例治**同一章内、同一对象的微观属性打架**；写完含"冷热/明暗/大小"对比的段落，建议逐句核对属性词是否互斥。

## 案例 #029 — 角色认知越界（让角色说出/想到超出其应知的信息）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-17，ch8 L73 林小月台词诊断）
- **现象（用户原话）**：「林小月忽然又呀一声，攥紧他：『先生，我腕子又热了，跟这门上的纹，像是一个来路。』说多错多，她现在就知道是一个来路了？」角色（尤其配角/同伴）脱口说出需特殊知识才能下的结论（同源/来历/机制/因果），超出其身份与当前所知的认知层级；且该结论往往是后文叙事者/主角才该揭示的设定关系，提前塞进角色台词既越权又抢悬念。
- **根因**：作者把"后文要揭示的设定关系"提前挂到角色嘴上，让角色成了设定的传声筒。角色台词/内心本应只含**其身份 + 当前所见所感**能推出的内容；涉及"同源/因果/来历/机制"等需特殊知识的判断，是叙事者或后文的职责，不是角色的。
- **错误写法（片段 ≤150 字）**：
  > 林小月攥紧他：「先生，我腕子又热了，跟这门上的纹，像是一个来路。」（村女只觉腕子烫，却一口诊断出"与门纹同源"——这是需特殊知识才能下的结论，且本该由后文叙事者揭示，她此刻根本不可能知道）
- **正确写法（片段 ≤150 字）**：
  > 林小月攥紧他：「先生，我腕子又热了，一挨近这道门，它就又烫起来。」（只说可观察的体感关联：靠近门→腕子发烫。不下"同源/来路"结论，把设定揭示留给叙事者/后文）
- **可固化规则**：
  - **角色台词/内心只含其认知层级内的内容**：写完后问——"这句话/这个念头，这个身份、这个时点的人**真的能说出来/想到吗**？"若涉及同源/因果/来历/机制/幕后真相等需特殊知识的判断，而角色并无此知识，即认知越界，必改。
  - **结论留给叙事者或后文揭示**：设定关系（如 A 与 B 同源、某印能动因果）的"盖棺"应由叙事者或具备相应知识的主角在合适时机点出，勿让配角/同伴提前当传声筒。
  - **用可观察内容替代结论**：角色能说的是**自己的体感、所见、所疑**（"腕子又热了""一挨近门就烫""这纹瞧着眼生"），而非对现象的本质定性（"同源""一个来路"）。把"为什么"留给学生/后文。
  - 与 #026（配角不预知主角该慢慢揭的谜底）、#028（内部属性自相矛盾）同源互补：#026 治配角替主线递**情节谜底**，本例治配角提前说出**设定关系/本质结论**；#028 治一句话内属性打架，本例治角色说了超出其层级的话。三者共同要求"信息分层——谁该知道什么，就只让谁说"。

## 案例 #030 — 角色情绪/状态缺乏前文因果（无来由的高兴/沉浸/兴奋）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-17，ch8 L95 林小月回程段诊断）
- **现象（用户原话）**：「回程的山路上，林小月蹦蹦跳跳，还沉浸在方才的深处……她在沉浸什么？还有为什么高兴？感觉很莫名其妙」角色被写成高兴/蹦跳/沉浸，但前文没有给出让她产生该情绪的**可溯源动因**，读者不知道她因何高兴，情绪显得凭空、悬浮。
- **根因**：作者直接给角色贴上情绪结果（蹦蹦跳跳/高兴/沉浸），跳过了"是什么经历触发了这情绪"的因果链。读者需要从前文找到角色**此刻状态的来由**，否则情绪就是无本之木。常见于：①用"沉浸在方才的 X"这类空泛短语概括情绪，不点具体触发物；②角色本章被动跟随、无正面体验，却被写成兴奋；③情绪与角色实际经历矛盾（刚经历寒意/凶险却蹦跳）。
- **错误写法（片段 ≤150 字）**：
  > 回程的山路上，林小月蹦蹦跳跳，还沉浸在方才的深处。陈言放慢脚步，回头望了眼洞口。……林小月哼起不成调的小曲，手却悄悄摸了摸自己腕子。（"沉浸在深处"空泛无对象；"蹦蹦跳跳"高兴无来由——她在秘境里只是被动跟随，前文未给任何让她兴奋的正面体验）
- **正确写法（片段 ≤150 字）**：
  > 回程的山路上，林小月脚步轻快，还惦着方才石门跟前腕子上那点热。先生说那热是门引的，她只觉新奇，原来自个儿身上也藏着这等名堂。……林小月哼起不成调的小曲，手却悄悄摸了摸腕子。（情绪动因落到具体可溯源的经历：腕子古纹被石门引动发热 + 一个村丫头发现"我身上也有奇处"的新奇感，与后文她跃跃欲试问"我也能说理不"连贯）
- **可固化规则**：
  - **角色情绪须有前文可溯源的动因**：写完一段带情绪/状态的描写，回问——"这个高兴/沉浸/兴奋，前文哪件事能解释？"若找不到，要么补出触发经历，要么把空泛情绪词（沉浸/高兴/莫名其妙地轻快）改成角色**具体在回想/在意的具体事物**（某句话、某个体感、某个发现）。
  - **避免"沉浸在方才的 X"式空泛概括**："沉浸"须接具体对象（沉浸在腕子上那点热 / 沉浸在那句碑文），而非"深处/方才/那片天地"这类无指称的空词。
  - **情绪与经历须不矛盾**：角色刚经历寒意/凶险/被动无助，就写蹦跳高兴会显假；若确因其发现了自身不寻常之处而小兴奋，须把"发现"这层动因写明，让其与后文态度（跃跃欲试/想自己站出来）形成连贯。
  - 与 #026（配角立体、有自己主线）、#015（动机能力相称）同源互补：#026 治配角动机不靠主线，本例治配角**情绪**也要有属于自身的来由；#015 治"能力是否撑得起行为"，本例治"情绪是否撑得起描写"。

## 案例 #031 — 凭空命名未定义地点/机构（造词风险）

- **来源**：告天 ch8 L99「我替你站上裁决台」，用户追问"裁决台是什么?"，查实全书正文与 world.md/rules.md/examples/geography 均无此词。
- **现象**：叙事或对话中出现一个具名地点 / 机构 / 设施名词，读起来读者"应该懂"，实则 world bible 从未定义；或与已存在的**反派机构撞名**（裁决台 ≈ 裁决司），造成逻辑矛盾。
- **根因**：写作时图顺手造了个"听起来像设定"的名词，未核对 world bible 的地理/场所表；长文多章时，早章埋下的新词往往到后期才被读者/作者自己发现。
- **错误写法（片段 ≤150 字）**：
  > 陈言屈指弹了下她额头：「……真遇着人欺你，先回来寻我，我替你站上裁决台。等你能自个儿立得稳那誓，再自己跟它说。」
- **正确写法（片段 ≤150 字）**：
  > 见 #032：换用 world bible 已定义的「刑台」只是第一步；但即便「刑台+向天道立誓」仍属对话里的机制说明，最终落地为不带机制名词的口语回话（"想得倒美…真有人欺你，先回来寻我。等哪天你自个儿立得住，再跟它说去"）。本例只解决"名词越界"，机制倒设定问题交 #032。
- **可固化规则**：
  - 写具名地点/机构前，先核对 world bible 已定义的地理/场所/机构清单；不在清单里的新名词，要么先补进设定再使用，要么改用已有词。
  - 具名场所须与角色身份/认知层级匹配：村姑懂"镇上刑台"，不必懂"无形公堂"；**反派机构的设施名，勿让主角主动把人带去**。
  - **机制优先于造台**：能用"向天道立誓 / 立誓请天道裁"描述动作，就不要另造一个"X台"承载该动作。
  - 与 #002（设定一致性）同源：#002 治"角色知识越界/出身矛盾"，本例治"名词越界/虚无地点"——都是写前未对齐 canon。
  - **自查**：写完后 grep 本章所有「台/殿/司/阁/府/门/窟/城/域」等具名场所/机构词，逐个确认是否 world bible 已定义；未定义且非有意为新设定的，改掉或补设定。

## 案例 #032 — 角色台词承载机制说明（对话变设定说明书 / 过度给方案）

- **来源**：告天 ch8 L99。上一版把"刑台+向天道立誓+立稳誓"三步流程当台词讲给林小月（村姑）听，用户指"怎么就要上刑台了，有必要说的这么细致吗"——即便 #031 已把"裁决台"换成已定义的"刑台"，这句仍别扭。
- **现象**：角色（尤其主角/长辈）对另一角色说出一段"步骤说明书"或机制解释，信息密度远超该对话场景所需，本质是作者借角色之口倒设定（info-dump）。常见形态：①给具体地点/仪式/流程（"你先去 X 台，向 Y 立誓，等立稳了再 Z"）；②给世界规则讲解；③把本该留白的机制一次性讲清。
- **根因**：作者心里想交代设定，又不想另起叙述段，便塞进角色台词；误以为"角色说了就更自然"。实则角色台词也须符合其当下动机与对话对象的认知层级，不是设定出口。
- **错误写法（片段 ≤150 字）**：
  > 陈言屈指弹了下她额头：「……真遇着人欺你，先回来寻我，我替你站上刑台，向天道立誓。等你能自个儿立得稳那誓，再自己跟它说。」
- **正确写法（片段 ≤150 字）**：
  > 陈言屈指弹了下她额头：「想得倒美。你今日敢挨近那道门，这股胆子就对了。真有人欺你，先回来寻我。等哪天你自个儿立得住，再跟它说去。」（砍光机制说明；"那道门"= 场景实存呼应她敢挨近的勇气，"跟它说"= 回声她"跟它说理"；"立得住"作"能自己撑住"的口语隐喻，不点机制）
- **可固化规则**：
  - 角色台词承载的信息须 ≤ 该对话场景与对象认知层级所需。对小孩/外行解释世界机制，给一句带温度的回话即可，勿给流程。
  - **判断标准**：删掉这句里的"地点/仪式/步骤名词"后，对话是否仍然成立且更自然？若成立，说明那些名词是作者想倒设定，删。
  - 世界机制优先用"场景展示"（碑面回应、古纹亮、他人反应）而非"角色讲"。与 #007（动作法/机制不裸奔）、#001（三删三加·说人话对话）同源，本例专治"对话里的 info-dump"。

## 案例 #033 — 跨章节时间线前向引用（把后章才发生/才登场的角色，当已发生/已见过来引用）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-17，复检 ch9 时发现「怎么就突然出现了一个玄阴宗」——追查下来是 ch9 把 ch12–14 的角色/事件当已知引用）
- **现象**：第 N 章（按文件序号在前）的叙述/台词/内心，引用了第 M 章（M>N，按时间线在后）才**首次登场**或**首次发生**的角色/事件，并当作"已经见过/已经发生"来写。典型三个伪装：
  1. 把未来事件当"已发生的旧案"列进主角的回忆/账册（如 ch9 把 ch12 才发生的"陆明死罪"写进钱万三的"对不上"旧案单，仿佛早已结案）；
  2. 用"当初/早先/那日"等时间词暗示主角"之前见过"某角色，但该角色在后章才初次见面（如 ch9"和当初陆明身上那道冤屈灰痕一个样"，可陈言在 ch12 才初见陆明）；
  3. 把后章才建立的"反派阵营/关系网"当成读者已知的前提来起手（如 ch9 开头"自打沈舟血案、陆明死罪两桩都缠着墨通那辈人的影子，但凡玄阴宗登门…"——陆明死罪未发、玄阴宗也未在正文登场过）。
- **根因**：AI 长程一致性缺口——写第 N 章时已"知道"全书后续设计（角色档案/案件库/总纲），无意识地把后章信息当成当前章的"背景常识"倒灌；且"用已知关系网做开头钩子"比"从当前章事件自然引出"更省力，于是跳过铺垫直接假设读者已懂。与 #002（设定一致性）同源，但是**跨章节时间轴**维度，比单章内设定错更难机器发现。
- **错误写法（片段 ≤150 字）**：
  > 秘境一行回来，陈言心里总搁着那半句残文，对玄阴宗的人越发留神。自打沈舟血案、陆明死罪两桩都缠着墨通那辈人的影子，但凡玄阴宗登门，他腕上灰线便先一步有了动静。（陆明死罪是 ch12–14 才发生的未来事件；"当初陆明身上那道冤屈灰痕"更暗示主角早见过陆明，但 ch12 才初见）
- **正确写法（片段 ≤150 字）**：
  > 秘境一行回来，陈言心里总搁着那半句残文：「司印所及，旧录可移」。钱万三旧档里那枚能改因果的印，执印人叫墨通，原是玄阴宗沈舟案的师兄。自那日起，陈言对玄阴宗的人便多留了分神。（用 ch8 残文 + ch4 已建立的"墨通=玄阴宗沈舟案师兄"线索把玄阴宗**接回已知**，L3 再让玄阴宗弟子在正文里**登场**；不引用任何后章才发生的事件/角色）
- **可固化规则（写前/写后硬性核查）**：
  - **时间轴自检**：写第 N 章时，凡引用某角色"已见过/早知道"、某事件"已发生/旧案"，先核对——该角色/事件是否在第 1…N 章**确实已登场/已发生**？若在 N 之后（按文件序号或剧情时序），即为前向引用，禁止。
  - **"当初/早先/那日/记得"类时间词**：凡用来指"主角过去见过 X"，X 必须在前章确实有见面/交代场景；否则删该词或改为当前章内新发生。
  - **开头钩子接法**：想用"反派阵营/关系网"做开头，只能接**前章已建立的线索**（如本例接 ch8 残文 + ch4 墨通），不得假设后章才建立的阵营为读者已知；后章才登场的阵营，应在当前章**正文里首次出现**（让角色/势力在场景里登场），而非在概述里当它已熟人。
  - 与 #002（设定一致性）、#024（符号卫生）、#031（幽灵名词）协同：前向引用常伴随"把后章名词当已知"与"概述起手"，三者一起查。
