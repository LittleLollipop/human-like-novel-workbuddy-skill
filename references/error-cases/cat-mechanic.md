# 错误案例 · 叙事机制/感知

## 案例 #006 — 设定符号过度复读（核心视觉化符号每次出场重建）

- **来源**：用户长篇连载项目反馈（全书，2026-07-11）
- **现象**：某核心视觉符号（文中以「线」具象化因果，以下用「线」作通用示例）在全稿出现 600+ 次，单章 24–66 次，读多了心烦。四类复读：①跨章重描（已出场角色的符号新章又完整描）②章内累进复读（第 N 个新符号把前 N-1 个的「引/感/承」全量复读）③固定系统反应（每次见新符号系统必动）④意象反复（「远方/大网」跨章复读）。
- **根因**：把核心设定的视觉化符号当成**每次出场都要重建**的信息，而非「首建、后续仅状态变化」。
- **错误写法（某章某角色金线段，同章内复读）**：
  > 第9行：这姑娘腕上也缠着一道线——清亮得像初春的溪，尾端浮着金…系统残念没动…连某角色甲那道银灰残念都认得个「远」字… 第17行：某角色甲银灰是「引」、某角色乙古纹是「感」、这姑娘金线是「承」，三道线三种力道，来自远方，比某反派那枚印大得多的网。
- **正确写法**：
  > 这姑娘腕上也缠着一道线，尾端浮着极淡的金——和前头见过的线都不是一路。主角心头微异，残念竟拿不准这金线的来路。（首次建立即可；后续该角色再出场只提「那道金线」；不回述引/感/承全貌）
- **可固化规则（设定符号降频纪律，详见 style-guide 第六节）**：
  - 首描详、复描略；不写累进对比段。
  - 系统/残念反应只在关键剧情点触发，非每次见符号都动。
  - 大意象（远方/大网）一次定性，不再抒情复读。
  - 破案必需的符号推理保留，只删四类复读冗余。
  - **用户二次反馈（关键修正）**：换字/减字频只是治标——读者烦的不是某个字多，而是**关于该符号的叙事节拍（看符号/想符号/符号亮）调用太密**。正确做法是**减少概念被调用的叙事节拍数**：删不推动剧情的冗余调用（章尾「符号亮了一息」、meta 总结「看因果一日比一日清楚」、情绪点「摸着那道亮符号」、多案密集罗列「看符号」），把空缺用案情/人物/世界生态填回。减字频（线→痕/因果）只能辅助，不能替代减节拍。

## 案例 #007 — 超自然感知「直白贴标签」（减了节拍仍出戏）

- **来源**：用户长篇连载项目反馈（2026-07-12）
- **现象（用户原话）**：「还是有点多，应该说是写的太直白了，你可以描写主角的动作，比如主角看了手腕，但是完全不提线和痕」
- **根因**：减了冗余节拍、换了字，但凡写到「感知」仍直接把超自然机制**标出来**（「灰痕一跳」「某红线连着某人腕上」「金痕拿不准」）。读者从首章已知主角有某种「看见」之能，后续无需每次点名机制；直接命名=唠叨且出戏。
- **错误写法（某章重做片段）**：
  > 主角一进堂，腕上灰痕便微微发沉——他看见，长老周身缠着一道灰红，自心脉起缓缓蚀着神识，那灰红的尾端，恰恰连在师姐腕上。
- **正确写法（动作+反应，不命名机制）**：
  > 主角的目光在长老和那师姐之间走了一遭。长老手抖得厉害，师姐递汤药的手却稳得很——稳得不像照料病人，倒像在数着日子。主角心里，那根弦无声地绷了一下。
- **可固化规则**：
  - 超自然感知**首章建立一次**后，后续章用「动作 + 对方可读取的神色 + 主角结论」传递，不重复命名机制符号（线/痕/灰红/金痕/古纹/银灰等）。
  - 写「他看了眼手腕」「凑近标本」「盯着手」「把腕子缩回袖子」等物理动作，让读者自行补足「他在用那双眼睛看」。
  - 案件章里机制符号若属**破案证据本体**（如某线=灵脉被截、某红线=下毒连线），可保留必要揭示，但**不要用「灰痕一跳/某红线连着」这类「个人反应 + fidget」反复贴标签**；证据靠「他凑近看 / 他盯着 / 对方脸色变了」呈现。
  - **不要硬加「被指引 / 被召唤」的被动神秘牵引（属 #012 事实诚实特例）**：主角走向某处、探查某物，通常有明确目的或主动反应，**不必写成「冥冥之中被什么牵引 / 指引」**。"寒意引着他""像被深处牵着"这类被动语态既混淆感知归因（见上），本身也是读者腻味的套路，应删。若需写身体对灵物 / 传承的感应，落在实际感知（传承发烫、识海异样）+ 主角**主动判断 / 目的**（"得去看看""总要到头看个明白"），不写"被X牵 / 引 / 指引"。错误：「越往里寒意越分明，像有什么引着他」（冷当引，归因错）；也错：「阴冷越重，体内传承发烫，像被深处牵着」（归因虽对，但"被牵"仍是被动神秘主义，依旧读者判假）；正确：「越往里阴冷越重，体内传承却渐渐热了起来。他没停步，既进了这石室，总得到最里头看个明白」（感知 + 主动目的，无被指引）。
  - 术语（如查线誓 / 看线 / 线亮）是机制名，可保留；叙事描写层一律不贴标签。
  - **实战漏点清单（ch9 复检，2026-07-18）**：以下均为"在叙事里命名主角/他人身上超自然符号"，须改。这些在首轮「全集回灌」时漏改，二次复检才清掉——自查时务必逐字扫，别只凭印象：
    - 「这人腕上缠着一道线，灰的，清的，没有半分盗的浊黑」→ 改为「陈言瞥了他一眼，便知这人干净——遭了冤、却辩不出的那种干净」（用「看」能力动词 + 结论，不命名线）。
    - 「把指尖轻轻搭在周野腕上那道灰痕上」→ 删「那道灰痕」，留动作「搭在周野腕上」。
    - 「周野的因果干干净净……没有半分作贼的痕」→ 「痕」改「因」（案术语，非符号名）。
    - 「腕上灰线亮起」→ 删，用动作/对白引出（如「陈言站起身，目光落在两人脸上：「立查誓吧」」）；发光效果留给后文「腕间光华大盛」统一呈现。
    - 「指了指周野腕上那道清灰痕」→ 「朝周野一指」。
    - 「他那道灰光便再亮一次」→ 「他这能看穿因果的本事，便再使一回」。
  - 与 #006 关系：#006 治「调用次数多」，本例治「即便次数少了仍直白」——两者叠加才是「读者不烦」的完整解。

## 案例 #019 — 同一视觉意象/设定词勿在相邻段落反复夹写（集中到关键处单独成段）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-16，某章开头通读）
- **现象（抽象）**：某核心设定视觉化符号（如角色特有的「感知线/因果线」类意象）在开头数段被反复提及——几乎每段都夹一句（「画面里人人都牵着线」「角色问：他们也有线？」「那缕线」「碑身缠着无数线」「这些线不是死的」「只看那些线便知……」），读者被同一意象刷屏，且每段往往只"提一句"而无新信息，读感烦。
- **根因**：作者担心读者记不住设定，于每个场景切换处都补一句意象提示；但设定视觉化只需在「第一次出现（画面）」和「情节关键处理处（集中解读）」两处落地，中间的过渡段不必每段重申。
- **错误写法（片段 ≤150 字）**：
  > 壁上人人腕上都牵着一根线。林小月问：「他们腕上也有线？」他答：「人人都有。」……像有根线在底下拽着他走。越往下那缕线越清晰，竟分出几股。……碑身缠着无数线。……这些线不是死的。……他只看那些线便知：千万缕缠成网……
- **正确写法（片段 ≤150 字）**：
  > 壁上人人腕上都牵着一根线，连向那截碑（首次画面，一笔带过）。……（下行过渡段不提线，只写体感/环境）……他眯眼，果见一缕极淡的冷线从深处漫出（异域线钩子，单独一段）。……（进石室仅画面一笔「碑身缠着无数线」）。……主角定睛去看那些线。它们不是死的：有的绷着有的缠错……织成一张网（集中大段，唯一的解读性描写）。
- **可固化规则**：
  - **设定意象两处落地即可**：①首次出现——用画面/动作一笔建立（读者看见即可）；②情节关键处理处——集中成一段完整描写（解读/作用/变化）。两者之间（过渡、环境、对话）不每段夹写该意象。
  - **同一意象勿刷屏**：若某章某一符号在连续多段都出现，检查是否每段都只"提一句"却无新信息——是则合并到关键段，过渡段用其他可感细节（环境/体感/动作）填充。
  - 与 #007（动作法，不裸奔机制）+ #016（脱节旁白，补感官不补交代）一致：意象靠场景画面与关键动作承载，不靠每段提醒式旁白。

## 案例 #022 — 超自然效果的「巧合式一闪」与 阴森/氛围段堆虚比喻

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-16，ch8 五修）
- **现象（抽象）**：两类"作者痕迹"重的写法——
  ① **超自然视觉效果写成"一闪即灭恰被主角逮到"的巧合**：某异象（金光/纹路/虚影）「浮起一层极淡的金，转瞬又灭，快得几乎像是错觉」，偏偏正好被主角这一瞬看见。读者会觉得"这么短的效果凭什么正好被逮到"，是作者硬安排的巧合。
  ② **阴森/幽暗/静谧的环境氛围段堆"虚比喻"**：写地下石室、密林、古墓这类氛围，连用「连呼吸都像凝住了」「静得反常」「仿佛踏入另个天地」等"像/仿佛+抽象喻体"的虚描写。喻体空泛、彼此同质，读来不但不吓人，反而假、腻、出戏。
- **根因**：①作者想"秀"一下神秘效果又怕过度暴露设定，于是让它"一闪即灭"——却制造了巧合；②以为氛围靠"比喻叠加"来渲染，实则抽象喻体越多越假，真正的氛围来自具体感官（光/声/温/湿/触）。
- **错误写法（片段 ≤150 字）**：
  > 那道古纹浮起一层极淡的金，转瞬又灭，快得几乎像是错觉。（巧合式一闪）
  > 四壁幽幽地照，连呼吸都像凝住了。满室静得反常，仿佛踏入另个天地。（氛围段堆虚比喻）
- **正确写法（片段 ≤150 字）**：
  > 她摊开手给他看，那道古纹泛着一层极淡的金，随腕子的热一跳一跳。（效果持续/由角色主动展示，主角看见顺理成章）
  > 四壁的光落在人脸上，泛出一层青。头顶那些喊杀声到这儿只剩闷闷一线，隔着厚厚的岩层，断断续续。（用光色+声音衰减落地，不用虚比喻）
- **可固化规则**：
  - **超自然效果避免巧合式一闪**：异象若需被主角观察到，让它**持续一段**、或由**角色主动展示/触碰引发**，而非"闪一下正好被逮到"。"快得像错觉"这类句式是巧合信号，见即改。
  - **氛围段慎用虚比喻**：写阴森/幽暗/压抑/静谧，优先用**具体感官**（光的颜色与强弱、声音的衰减/回响、温度、湿度、气味、触感）落地；少用"像/仿佛/仿若+抽象喻体"（凝住/另个天地/反常）。每个氛围段虚比喻至多一处，且须有具体载体支撑。
  - 与 #007（动作法，不裸奔机制）、#012（事实诚实）一致：氛围与超自然都靠可感的具体呈现，不靠抽象喻体或作者安排的巧合。

## 案例 #027 — 「刻在心里／暗中记下」式默记线索套路（信息处理偷懒句）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-17，ch8 线索处理句诊断）
- **现象（用户原话）**：「只把这几个字刻在了心里。不要总是刻在心里、暗中记下什么的」。主角每次得到线索/听到关键信息，作者就用「刻在心里／记在心里／暗中记下／默默记下／把这笔账记下了」一类句子收尾，通篇高频复用，成了处理线索的万能套路句。
- **根因**：作者用一句固定的"内心归档"动作替代真正的人物反应，回避写"这条线索对他意味着什么、他接下来的态度/取舍是什么"。它既是**复读**（同一句式反复出现），也是**信息处理偷懒**——把"角色如何消化信息"简化成一个仓库入库动作，读者看不到人物的判断、情绪或选择。连带地，配角的记仇/记情也常被写成同一个「记下了」，加剧重复。
- **错误写法（片段 ≤150 字）**：
  > 陈言没接腔，只把这几个字刻在了心里。……（隔几段）这笔账，王虎记下了。……（再隔几段）「陈讼师，这一程，小人记下了。」（三处「记下」高频复现，主角对线索、反派对仇、配角对情分全用同一个入库动作，无一处写出具体态度/情绪）
- **正确写法（片段 ≤150 字）**：
  > 陈言没接腔。那半句残文他看过一遍就够，管事要不要掌眼是另一回事，他没应承。……这笔账，他咽不下去。……「陈讼师，这一程多承照应。」（主角落回"取舍/态度"，反派落回"情绪"，配角落回"谢辞"，各写各的具体反应，不再共用「记下」）
- **可固化规则**：
  - **禁止用「刻在心里／记在心里／暗中记下／默默记下／(把某事)记下了」作为线索/信息的处理收尾**，尤其禁止全文高频复用。得到线索后，写角色对它的**具体态度、判断或取舍**（信不信、要不要用、和已知的哪条对上、下一步做不做），而非"存进心里"这个无信息量动作。
  - **区分保留**：字面动作义的"记下"可留（如"他随手记下的猜想"描述某人写笔记的物理动作、"记账/记名字"等实务），要清的是**用作心理归档套路**的那类。
  - **配角的记仇/记情也别都写「记下了」**：记仇→写情绪（咽不下、忍不下、盯着某人背影）；记情→写谢辞/回礼意向（多承照应、来日必报），避免与主角的线索处理句撞成同一个词。
  - 与 #006（符号/词语过度复读）、#020（套路化句构）同源：本例是"信息处理"场景下的专用套路句。
  - **情报/记账类角色"即时归册"是具象动作，非本套路**：如钱万三商会情报角色当场把线索"归到蓝皮册同页"、账房"入账"——这是该角色自身主线的具体行为，与"主角/配角把线索往心里存"的偷懒句构不同，保留。判别点：是否落在可观察的外部动作（翻册/落笔/归档）而非"心里/心底"内部入库。
  - **写完一章必用全覆盖 grep 自查**（模式不全＝假绿，会漏网，见下"坑"）：`刻在.*心里|记在.*心里|记在心底|压进.*心底|压在.*心底|藏在.*心里|藏在.*心底|收进.*心底|收在.*心底|搁在.*心里|存进.*心里|沉进.*心底|暗记|暗自记|默默记|暗中记|记下了`，命中须逐处判是否"心理归档套路"，>1 处即换写。注意「记在心里」「压进心底」等变体缺一不可，首轮若只 grep 部分变体（如只查「记下了/刻在心里」）会漏掉「记在心里」「压进心底」类写法——这正是实测漏网根因。

## 案例 #032 — 角色台词承载机制说明（对话变设定说明书 / 过度给方案）

- **来源**：告天 ch8 L99。上一版把"刑台+向天道立誓+立稳誓"三步流程当台词讲给林小月（村姑）听，用户指"怎么就要上刑台了，有必要说的这么细致吗"——即便 #031 已把"裁决台"换成已定义的"刑台"，这句仍别扭。
- **现象**：角色（尤其主角/长辈）对另一角色说出一段"步骤说明书"或机制解释，信息密度远超该对话场景所需，本质是作者借角色之口倒设定（info-dump）。常见形态：①给具体地点/仪式/流程（"你先去 X 台，向 Y 立誓，等立稳了再 Z"）；②给世界规则讲解；③把本该留白的机制一次性讲清。
- **根因**：作者心里想交代设定，又不想另起叙述段，便塞进角色台词；误以为"角色说了就更自然"。实则角色台词也须符合其当下动机与对话对象的认知层级，不是设定出口。
- **错误写法（片段 ≤150 字）**：
  > 陈言屈指弹了下她额头：「……真遇着人欺你，先回来寻我，我替你站上刑台，向天道立誓。等你能自个儿立得稳那誓，再自己跟它说。」
- **正确写法（片段 ≤150 字）**：
  > 陈言屈指弹了下她额头：「想得倒美。你今日敢挨近那道门，这股胆子就对了。真有人欺你，先回来寻我。等哪天你自个儿立得住，再跟它说去。」（砍光机制说明；"那道门"= 场景实存呼应她敢挨近的勇气，"跟它说"= 回声她"跟它说理"；"立得住"作"能自己撑住"的口语隐喻，不点机制）
- **可固化规则**：
  - 角色台词承载的信息须 ≤ 该对话场景与对象认知层级所需。对小孩/外行解释世界机制，给一句带温度的回话即可，勿给流程。
  - **判断标准**：删掉这句里的"地点/仪式/步骤名词"后，对话是否仍然成立且更自然？若成立，说明那些名词是作者想倒设定，删。
  - 世界机制优先用"场景展示"（碑面回应、古纹亮、他人反应）而非"角色讲"。与 #007（动作法/机制不裸奔）、#001（三删三加·说人话对话）同源，本例专治"对话里的 info-dump"。

## 案例 #042 — 机制过度延伸 / 超自然机制逻辑不自洽（把标记/线接到读者无法理解如何连接的符号上）

- **来源**：2026-07-18 ch9 L21。原「那道银红自草身起，连向玄阴宗灵库的方向，再往深处，竟缠上一枚陈言极眼熟的印痕，裁决司的旧印。」用户问：「居然还能牵扯到印，这印是盖在草上吗?」——指出银红（草上被按上去的红记）被写成"缠上一枚印痕"，等于把一条因果线凭空接到一个飘在别处的印章上，读者既看不到印在草上，也想不通两样东西怎么连上，机制断裂。
- **现象**：超自然能力（看因果/追线）的描写中，作者把 A 标记/线**延伸接到** B 符号（印痕/人名/地点），却没建立"如何连"的逻辑，导致读者要追问"这俩是怎么扯上关系的"。常叠加"再往深处""极眼熟的"等填料，越伸越虚。
- **根因**：AI 把"能力可以追线"当成万能胶，以为写"缠上/连向/牵出"就能自动把任意两个设定焊在一起，不校验读者是否能在文本内理解这条连接的因果。
- **错误写法（片段 ≤150 字）**：
  > 那道银红自草身起，连向玄阴宗灵库的方向，再往深处，竟缠上一枚陈言极眼熟的印痕，裁决司的旧印。
  > （印没盖在草上，线却"缠上印痕"——读者问"印是盖在草上吗"，答不上来即断裂）
- **正确写法（连接须内生于已呈现之物：要么标记本身即该符号的残痕，要么用可见线索搭桥）**：
  > 那道银红自草身起，连向玄阴宗灵库的方向，而那红记本身的纹路，分明是一枚印的残痕：裁决司的旧印。
  > （不"缠到"印；是红记**自身纹路**即那枚旧印的残痕——印证墨通旧印改过因果、红记是其手笔；印的痕迹本就在草上这红记里，逻辑自洽）
- **可固化规则**：
  - 写完超自然追踪/连接后自问：`读者能在文本内理解 A 与 B 是怎么连上的吗？` 若连接靠"缠上/牵出"这种万能动词悬空，则改为**内生于已呈现之物**（A 的纹路/性质即 B 的残痕/签名），或补一条可见线索搭桥。
  - 不写"再往深处/遥遥连向/极眼熟"等把连接越推越虚的填料；连接一旦成立，点到即止。
  - 与 #007（动作法·不裸奔机制符号）、#012（机制不堆砌）、#038（视角不未卜先知）关系：本例治"机制连得通不通"，是能力描写的**逻辑闭环**维度，与上三者互补。

## 案例 #051 — 章中漂浮感官句（无锚点 / 指代悬空 / 甩尾单字）

- **来源**：2026-07-20《告天》ch10 L3「这股沉味，他在镇上老衙门的档案室里也闻过，是时间堆出来的，沉。」用户：「不前不后的很怪异，这个他无法理解，最后的沉也不像人话」。
- **现象**：章中插一句感官/描写，却无锚点（前后都是环境或别人的戏，读者不知谁在感知）、代词无 antecedent（「他」前文没出现任何人）、结尾甩一个单字形容词当独句（「沉。」），不像人话。
- **根因**：作者想加氛围但没把感官细节挂到「当下正在感知的某个角色」身上；代词指谁没就近交代；为显文青甩个裸形容词收尾。
- **错误写法（漂浮）**：
  > 户籍堂坐落在镇中央……墙上挂着历代族谱匣，黑沉沉一排。（下接）这股沉味，他在镇上老衙门的档案室里也闻过，是时间堆出来的，沉。（上句纯环境无「他」，「他」无着落；甩「沉。」）
- **正确写法（挂到主体）**：
  > 户籍堂坐落在镇中央……脚下的青石板被无数双脚磨出了凹，踩上去微微发响。陈言把这些都看在眼里。堂里一股陈味也钻进他鼻子，旧木混着几十年积灰，跟他早年替人翻卷宗时待过的衙门档案室一个样。（感官挂到「陈言」这明确主体，无甩尾）
- **可固化规则**：
  - 写感官/环境细节前自问：`这句谁在感知？` 若答不出具体角色，或就近前文无该角色，先把主体点明（陈言看在眼里 / 他迈进堂）。
  - 代词（他/她）须在前一两句内有明确 antecedent；漂浮句最易「他」悬空，发现就补主体或并入有人的段落。
  - 结尾绝不甩单字形容词当独句（沉。/冷。/空。）；要收也收在完整分句里。
  - 与 #007（超自然感知动作法）、#047（描写失真）同源：感官要「长在角色当下动作上」，不裸奔、不飘。

## 案例 #052 — 非主角感知因果线（违反「因果线视觉化 = 主角专属」canon）

- **来源**：2026-07-21《告天》ch10。L11 周户籍官（凡人书吏）「这孤女腕上金线亮起的一瞬」；L41/L59 林小月「腕上那根黑线粗得吓人」「那林鹤腕上黑线沉得压人」。用户：「大部分人应该是看不到线的吧?」
- **现象**：非陈言的角色在「看/感知因果线」（金线/黑线/灰线/血脉纹），但按 canon（settings/rules.md §三「因果线视觉化（主角专属能力）」），因果线是陈言独有能力，普通人看不见。
- **根因**：作者忘了「线」是主角专属；唯一例外是「验线铜盘 / 天道显像」等法器仪式让线**公开显形**（在场人人可见），此外任何人（含弟子林小月）都看不见线。
- **错误写法（破 canon）**：
  > 周户籍官……可今日这孤女腕上金线亮起的一瞬，他想起自己那早夭的女儿。（凡人书吏看见金线 = 破 canon）
  > 林小月攥着陈言袖口：「那林鹤腕上黑线沉得压人，您可别硬顶。」（弟子也看见黑线 = 破 canon）
- **正确写法（改寻常可见）**：
  > 周户籍官……可今日这孤女攥着衣角立在堂中，眼圈红着，腰背却挺得笔直的一瞬，他想起自己那早夭的女儿。（触发点改成任何人可见的勇态，不依赖超能力）
  > 林小月攥着陈言袖口：「那林鹤脸色沉得压人，您可别硬顶。」（改成可见的「脸色/气场」，非线）
- **可固化规则**：
  - 凡写某角色「看见/感知一条线（金/黑/灰/血脉纹）」，先问：`这是陈言，还是有验线铜盘/天道显像等公开仪式？` 两者皆否 → 必错，改成该角色**寻常可见**的东西（脸色、气场、姿态、身上可见的胎记式古纹）。
  - 林小月腕上「古纹」是**可见胎记**（林鹤曾盯她腕子），非因果线；写他人对古纹的反应用「看见那道纹/忌惮」，不用「看见线」。
  - 仪式公开显像（铜盘垂落金纹、天道显像铺开画面）在场人人可见，可写众人反应，但须点明是「法器/天道显形」语境，不写成某人私能。
  - **四类延伸越界（ch10 后续实测，均属本例）**：
    1. **金线进带引号的当庭陈词**：陈言当着林家面用引号念「这金线从她爹、她爷爷连到上三代，根根不断」——把专属视觉当众宣告，且重复自身观测。应删，让论据靠「血脉因果一验便知」推进。
    2. **金线进对第三方陈述 / 第三方目击动作**：「林家数代与她的金线连着还是断了」「他伸手虚指那片金线」——第三方听不到也看不见线，越界。改为公共事实（血脉因果）或仅虚指腕上。
    3. **无铺垫的专属视觉线**：「那根连着主支权位的黑痕，微微颤了颤」——元素全书从未定义，且接在客观反应后读来像公共事实，越界+杜撰。删，用客观反应（脸色变）承载。
    4. **第三方确知主角专属视觉内容**：林鹤「被那讼师当堂看穿」——金线陈言独见，林鹤只看见陈言可见动作（盯腕子/请验血脉），只能推测「陈言冲那印来的」，不能确知「看穿」。写推测不写确知。
  - 与 #002（设定一致性）关系：本例是 canon 级的「能力归属」矛盾，写完可顺手核：本章每条被「看见」的线，是否都落在陈言或公开仪式上；另与 #029（认知越界）、#038（未卜先知）联动——凡人「看见/确知」主角专属视觉即越界。

## 案例 #054 — 客观状态可第三人称旁白，但禁人物想法/历史/主角推测（推测写法最烦人）

- **来源**：告天 ch10 周户籍官段，用户对「锁主角 POV + 陈言推断配角心理」的返工稿再否定：「还是第三视角在讲故事」「总是以主角的角度看太单调，而且这种推测的写法是最烦人的。与其这样还不如不写。对于客观的人物表现，状态可以用旁白的第三视角来写，但是不能夹杂人物的想法，人物的历史等等。这个早夭女儿、米缸碎之类的背景完全可以在后续的对话中写」。
- **现象（用户原话）**：① 总以主角角度看太单调；② 让主角**推测配角内心**（「陈言当讼师那些年这类老吏见得多了…这人不是糊涂是怕」「陈言也看出这老头的怕里掺着别的…他年轻时未必没梗过脖子」）最烦人；③ 配角的想法、历史（早夭女儿/米缸碎）不该在旁白里倒。
- **根因**：上版 #054 把「锁主角 POV + 主角推断配角心理」当正解，但**主角推测配角内心**恰恰是用户点名最烦的写法——它既单调（永远从主角脑壳往外看）又令人生厌（作者在替配角做心理透视）。真正的解法不是换摄像机位置，而是**分层**：可观察的「状态/行为」用第三人称旁白直写（既破单调又干净）；人物的「想法/心事」与「历史/背景」一律不进旁白。
- **错误写法（主角推测配角内心 / 旁白倒背景）**：
  > 陈言一进堂便见周户籍官翻册的手停了一瞬…这不是没看见是不敢看。陈言当讼师那些年这类老吏见得多了，手上那点权只够低头盖印。这人不是糊涂是怕。陈言也看出这老头的怕里掺着别的，年轻时未必没梗过脖子替人争过公理…陈言看他喉头动了动，心里有了数，这满堂认权的人里藏着一个偷偷盼他把理争回来。（主角对配角内心的推断 + 配角写实往事，读者最烦）
- **正确写法（客观状态旁白，无想法/无历史/无推测）**：
  > 户籍官姓周，是个干瘦老头，眼泡肿着。他翻册的手停了一瞬，又翻得更快，眼皮却始终不抬。那本该他经手的户籍册，早被林鹤的人抢先摊在了案上。他翻册的当口，眼角总往堂中那孤女身上瞟，瞟一眼又赶紧收回，握着铜盘边的指节泛了白。喉头动了动，终是没出声，只把目光往林鹤那边瞟了一眼，又垂下。攥着铜盘的手，悄悄松了半分。（全是第三人称可观察的行为/状态；不写他想什么、不写他过去、不写主角推测）
- **可固化规则**：
  - 写配角时**分层**：**状态/行为（可观察）** → 第三人称旁白直写，合法；**人物想法/心事** → 不进旁白（删，或留待该角色**自己对话**吐露）；**人物历史/背景（早年往事、家破、丧亲如「早夭女儿」「米缸碎」）** → 不进旁白，留待该角色**后续对话**中自然带出。
  - **主角推测配角内心 = 最烦写法**，宁可少写甚至不写，也不要让主角替配角做心理透视。「陈言想这人多半…」「他年轻时未必没…」「心里有了数，这人偷偷盼…」这类推断句一律慎用/删。
  - 纯主角限知视角易单调；用**第三人称客观旁白写配角可观察状态**可在不越界的前提下增加视角层次，但红线是「不写配角想法、不倒配角历史」。
  - 本例整合 #053 的 Tell→Show 要点：背景/状态可用第三人称旁白**具身呈现**（动作/状态），但摄像机须架在「读者该跟随的视角」上；非视角人物的想法、历史、被主角推测的内心，一律不进旁白。
  - 与 #007（动作法：状态用动作显）、#014（少写内心）、#026（配角有自己主线，背景靠其自身行动/对话带出）、#053（Tell→Show 错误示范史）联动；本例进一步限定：Tell/Show 之上还有「想法/历史禁入旁白、主角不推测配角内心」一层。
