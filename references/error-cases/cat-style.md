# 错误案例 · 文风/修辞

## 案例 #001 — 故弄玄虚式文风（系统性，最致命）

- **来源**：用户长篇连载项目反馈（2026 年初，前 5 章）
- **现象（用户原话）**：「少读几段时没有感觉，读多了就感觉故弄玄虚，不好好说话，很难一直读下去」
- **根因（三个叠加，非单章硬伤）**：
  1. **信息空转**：同一信息换种说法反复讲（如某角色进门「银灰线+残念」原样说两遍；章尾「远方的线」连说 5 次；「某角色古纹能感应」绕 4 遍）。读者潜意识捕捉到「故事原地打转」便烦躁。
  2. **章尾无限抒情拖尾**：正戏结束后又接 20+ 行意象堆叠感慨，每章如此，读者每次到章尾都得熬一段作者独白。
  3. **文言虚词密度过高**：通篇端着，没有一句大白话做缓冲，像全程拿腔拿调。
- **错误写法（片段）**：
  > 腕上灰线忽地一跳——这年轻人腕上也缠着一道线，尾端泛着极淡的银灰，像从千万里外一路缠来……识海里的残念，极轻地动了一息，似认得那银灰的来路，却又不全认得。（同一信息在第 5、9 行原样复述）
- **正确写法（片段）**：
  > 主角抬眼打量，腕上灰线忽然跳了一下。这年轻人腕上也缠着一道线，尾端泛着极淡的银灰，像从很远的地方缠过来。识海里那点残念动了动，仿佛认得，又不全认得，只留下个模糊的「远」字。（信息只说一次）
- **可固化规则（三删三加）**：
  - **删复读**：一个点只说一次，关键处最多呼应一次。
  - **删同义堆叠**：同一层意思用 2-3 个同义词叠说（「不疼不痒/只当胎记/省得瞎操心」「连着上古/连着很远/连着一处没资格碰的源头」）也属复读，留一句即可；让步理由须与角色真实动机一致，不能为了工整硬凑三层（ch10 实测）。
  - **删章尾抒情拖尾**：正戏结束即收，压到 2–3 句，落在动作或钩子上，不铺意象。
  - **删空转虚词**：删「半晌/足见/端的」等无信息量虚词。
  - **删过度修辞/浮夸（长篇小说最高频，详见 #064）**：角色内心独白、章节收尾句、出场句禁用文艺比喻与虚词堆叠；轻修饰（「压了多年」「那桩」刻意量词）也属浮夸；能用平实动作/对话传达的不堆比喻。
  - **删无信息量总结/复述（详见 #064）**：前文已坐实的信息（某手法相同、某伏笔已埋），禁止后续用角色内心/作者旁白再完整复述；角色内心只承载本段新推到的下一步。
  - **加平实缓冲句**：文言腔之间插入大白话短句，松紧交替。
  - **加说人话的对话**：对话带急、顿、噎，别都「不疾不徐」。
  - **加具体动作细节**：用动作/感官代替抽象感慨。

## 案例 #011 — 把生造表达包装成"化用/修辞"给合法化

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-13）
- **现象**：作者（或分析者）把一个**生造的具象表达**当成"化用成语/修辞妙笔"来正当化，掩盖它其实不合规范。如把「刻在骨头上」讲成"化用刻骨铭心的具象化改写"——既错用"化用"术语（无读者可辨识的源文本），又给生造镀了金边。
- **根因**：混淆"生造"与"合理新意象"；习惯给不规范表达贴修辞术语使其显得高级。
- **错误写法（认知）**：
  > 称「刻在骨头上」为"化用刻骨铭心、具象化改写成语"——错：化用要求源文本可辨识，拆成语当面说不满足；它只是个生造的直白夸张。
- **正确写法（认知）**：
  > 「刻在骨头上」是建在「刻骨」语素上的直白具象夸张/比喻，谈不上高级技巧；它是生造表达，只因生造得合乎汉语直觉、读者秒懂才勉强可用，绝非"化用/妙笔"。
- **可固化规则**：
  - **"化用"有硬边界**：必须存在读者可辨识的源文本，改造在合理范围。把成语拆开随便摆、读者认不出母本，不是化用。
  - **生造分好坏，不镀金**：坏生造=破坏汉语直觉、谁都卡壳的词组（如"教你说见""更慢的东西"）应删；可接受具象新意象=建在已有语素/意象逻辑上、读者秒懂（如顺"刻骨"逻辑展开）。关键是**别把生造包装成化用/修辞妙笔**——生造须受"读者秒懂 + 合乎汉语意象"约束，做不到即病句删。
  - 与 #010 关系：#010 治"用词撞语用限制"，本例治"为生造表达乱贴修辞术语"——两者都要求写手对词语保持诚实，不靠术语给不规范表达开脱。

## 案例 #012 — 用词与场景的"事实诚实"五类坑（脱离事实夸张 / 坏生造词 / 逻辑·场景·指代·重复错 / 错扣 canon 专名 / 机制隐喻硬收）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-13 多轮通读）
- **现象**：五类问题独立但同根——写手对"词语 / 场景 / 设定"不够诚实，靠夸张、生造、错位、机制裸奔来盖过事实或硬造钩子。
- **根因**：下笔前未核对"剧情给定事实基底"与"词语使用条件"，把"读着还行"当成"对"。
- **五子类（错误 → 正确）**：

  **① 脱离事实的夸张**
  - 错误：「一个老人对自己昏聩半生的愧」——角色实则是**近期**被下毒才神识昏聩，并非半生。夸张超出剧情事实基底，读者判假。
  - 正确：「一个老人，对自己昏聵、错怪了人的愧」——落到具体事实。
  - 规则：**夸张须有事实地基**；一旦脱离剧情给定的时长/程度/因果，读者立刻觉得假。

  **② 坏生造词（"似词非词"）**
  - 错误：「露出底下枯、无用的本记」「已涉故意慢害师长」——"本记""慢害"不是汉语词，是拼错的似词非词，读者秒噎。
  - 正确：「露出底下枯、无用的本相」「已涉故意谋害师长」。
  - 规则：坏生造=破坏汉语直觉、谁都卡壳的词组（本记/慢害/教你说见/更慢的东西）应删；可接受具象新意象须合汉语逻辑、读者秒懂（见 #011）。

  **③ 逻辑·场景·指代·重复错误**
  - 错误：「某角色这眼，和那是同一个。」（两人不能共有同一眼睛/眼神；"这眼"指代不清）；「他转身出堂……陈言没打扰，只把这一幕收进心底」（人已出堂又"在堂内看"）；室内写「老泪混着雪沫」（屋里无雪）；同一场和解写两遍（先写认错、后文又写"扶起"，与"已起身"矛盾）。
  - 正确：「某角色这眼神，和那是同一种。」；删提前的出堂+雪景，收束留在堂内；重复和解只留一次。
  - 规则：整句/段过"逻辑自检"——两人能共享此器官/神情吗？场景（室内外、在场离场）与动作自洽吗？同一事件写了两遍吗？

  **④ 错扣 canon 专名 / 错引台词**
  - 错误：本章某案是师徒私事（与某宗门无关），却写「某宗门因他立的誓免了一场死伤」，又引了属别章的台词「照看某宗门」——案件归属错、时间线错、无中生有。
  - 正确：本章只收尾本案件的真相亮明；该宗门/该台词移交其归属章节，不在此硬塞。
  - 规则：专名/台词须严格按 canon 的"所属案件/时间线/人物关系"使用；跨章借用须核对归属，不把别章钩子焊进本章。

  **⑤ 超自然机制隐喻硬收（无铺垫强行钩子）**
  - 错误：章尾「某域水面下那些缠着的因果，今日又分明了一分，像是被人从水底，又拨亮了一缕。」——诗化外衣裸奔核心机制（视觉化符号），两个"又"悬空前文无承接，"分明/拨亮"语义重复；与 #007 不裸奔机制冲突。
  - 正确：删机制隐喻，改为基于本案件事实的角色经验判断——「这一案，某人昏聩、某人下毒、某宗门上下噤声，分明是有人生怕真相见光。主角走南闯北这些年，见过太多这样的遮掩，知道幕后的手脚，往往比台面上的礼数更阴。」
  - 规则：钩子基于剧情事实 + 角色经验/判断来埋，不靠超自然机制的隐喻（水底因果/拨亮一缕/网清晰）强行收束；埋钩子 ≠ 裸奔机制。

- **可固化规则（总）**：写词句/场景前三问——①这词有人称/语体/事实限制吗？（#010/#011）②脱离剧情给定事实了吗？（夸张/生造/错扣）③场景动作逻辑自洽吗、机制裸奔了吗？（#007）诚实面对"剧情事实基底"，不靠夸张/生造/错位/机制隐喻盖过或制造。

## 案例 #017 — 叙事层内心总结/作者点题感慨裸奔（同一件事动作+点评写两遍）

- **来源**：用户长篇连载项目反馈（修真题材，2026-07-16）
- **现象**：一段里先写角色用动作完成某事（捡了某物/做了某取舍），紧接一句「这下 X 够了」「小人物的 X，够用便好」「这点东西上不得台面，却是一份实打实的进项」之类的内心总结/作者点题感慨。读者读到作者借叙事口吻替角色把潜台词喊出来，且同一信息动作里已呈现一遍、感慨里再来一遍 = 双重啰嗦、出戏。
- **根因**：作者不放心读者自己接，非要借叙事层补一句点题。但小说靠呈现不靠点评；动作已承载意思，感慨是冗余复述。
- **错误写法（片段 ≤150 字）**：
  > 他指尖一掐，连根带土塞进怀里，又顺手捞了块碎石料，在掌心掂了掂。分一点机缘，够了。他将草与石贴身收好，又借乱瞥了眼石窟——灵光更盛。他没凑那热闹，小人物的机缘，够用便好。指尖还沾着草根的湿泥，他拢进袖里——这点东西上不得台面，却是一份实打实的进项。
- **正确写法（片段 ≤150 字）**：
  > 他指尖一掐，连根带土塞进怀里，又顺手捞了块碎石料，在掌心掂了掂，草石收好，指尖还沾着草根的湿泥，随手拢进袖里。又借乱瞥了眼石窟——灵光更盛，几道身影正往里挤，他没凑那热闹。
- **可固化规则**：
  - **叙事层不裸奔内心总结/作者点题**：角色做完动作/选择后，删紧随的「这下够了/够用便好/实打实进项/说到底就是 X」类感慨句。意思已由动作承载，感慨是冗余+出戏。
  - **章尾 / 场景尾不承载全书级伏笔总结或「看似 X 实则 Y」式升华**：把「那些被改过的裁决」「东域家族暗里连着同一根源」「今日这一胜看似翻了铺面，实则撞动了某根弦」「压住的从来不是铺面，是这纹背后的东西」这类跨章大铺垫，留给后续章节分散落地，不在本章尾一次性倒给读者（ch10 实测）。
  - **无信息量过渡段应整段删而非精简**：某段读者已从上下文掌握其全部信息、且本身不烘托氛围（如主角内心过渡「看着护院匆匆离去，林家之上怕还压着人，把疑窦压心底」），删掉后读者零损失 → 直接删，不要精简成短版（短版仍可能无信息量+不烘托）。
  - **同一件事不写两遍**：动作流呈现过的（捡了草石=分了机缘），不要再发内心总结复述；呈现与点评分居，不要两遍。
  - 与 #009（角色台词补总结尾巴）同根——#009 治台词里，本例治叙事层；与 #014（少内心独白）互补——#014 治长段心理推演当发动机，本例治短促感慨/点题句；与 #016（脱节信息交代）同源但本例特指贴着场景却画蛇添足发感慨复述；与 #001（三删三加·删章尾拖尾）联动。

## 案例 #020 — 套路化句构「X了一分」与「人物台词塞旁白描写」

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-16，ch8 通读 + 跨章扫描）
- **现象（抽象）**：两类偷懒写法反复出现——
  ① **固定量化句构「X了一分 / X了几分」**跨章反复当"心理/氛围推进器"用（「心里那点疑，又沉了一分」「那根绷着的弦，又紧了一分」「把他心底那口井的轮廓，又描清了一分」），读者一眼认出是作者的习惯动作，且每章都来一下，读感腻。
  ② **人物台词里塞 narrator 的视觉/比喻描写**：让角色用 narrator 的紫腔形容自己身上的东西（「腕子上那旧纹……它自己热起来了，金闪闪的，像要浮出来」）。真实的人不会用「金闪闪的，像要浮出来」描述自己胎记——这是把本该由 narrator 呈现的感官细节，借角色之口说出，既假又越界。
- **根因**：①图省事，用固定增量句推进心理/氛围，缺变化；②把"呈现感官"误当成"角色能说的话"，角色台词边界失守。
- **错误写法（片段 ≤150 字）**：
  > 他记起外头誓石也认过她，心里那点疑，又沉了一分。
  > 她点头：「腕子上那旧纹……它自己热起来了，金闪闪的，像要浮出来。」
- **正确写法（片段 ≤150 字）**：
  > 他记起外头誓石也认过她，垂下眼。这丫头的来历，怕不简单。
  > 她点头：「那旧纹……自己热起来了，烫得慌，一跳一跳的。」（人物只说真实体感，视觉/比喻留给 narrator）
- **可固化规则**：
  - **忌固定量化句构**：心理/氛围推进禁用「X了一分 / X了几分」套构当习惯动作；每章至多一次，且优先用**具体动作/状态描写**承载（「垂下眼」「绷到了极处」「描清了轮廓」→ 改「把那口井的轮廓，又看清了一层」或干脆用可见事态）。自检时全局搜「了一分 / 了几分」，超 1 处必改。
  - **人物台词只说角色当场能说出口的实话**：体感（烫/痛/慌）、情绪、信息。不得替 narrator 做描写（颜色词、比喻、外形评价）。视觉/比喻/外形交给 narrator 在台词之外呈现。
  - 与 #006（复读）同根——#006 治词面重复，本例治**句构套路化**与**台词越界**两类；与 #007（动作法，不裸奔机制）、#017（叙事层总结）互补：都是"作者偷懒用固定套路替代具体呈现"。

## 案例 #023 — "Noun一Verb" 四字结构（重复率高的廉价叙述节拍：含体内 tell 与体外事件 cliché）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-16，ch8 六修 + 七修）
- **现象（用户原话，两批合并）**：
  ①「猛地一热 虽然不是x了一分，但是感觉是一样的，这种写法你还有很多，我觉得你应该用一来判断是否出现这种写法」——体内突发体感 tell。
  ②「我不同意你要保留的碑身一震/虚影一颤/台心一颤这些所谓的外部事件或动作，你完全可以换别的词语，没有四个字的就用一句话描写，非要写这种重复率极高的东西出来吗？」——即便是"外部事件"，凡"名词一动词"四字模板（碑身一震、虚影一颤、台心一颤、脚步一顿、脚下一顿、脸一沉、一字一顿、光影一晃、齐齐一颤）也属重复率极高的偷懒写法，不能保留。
- **根因**：作者用一句固定四字模板替代具体描写——要么把情绪外化成"体内猛地一X"的短 tell（常伴随来路矛盾），要么遇到任何"动作/事件停顿"就套"Noun一Verb"（碑身一震、脚步一顿、脸一沉），全书的转折与反应全靠这几副面孔，重复率极高、毫无新意。
- **错误写法（片段 ≤150 字）**：
  > 碑身一震，体内那点传承猛地一热，一股暖流顺着小臂涌上来，像游子归了巢。（体内→小臂来路矛盾 + 猛地一热 tell + 碑身一震 事件 cliché）
  > 陈言脚步一顿，余光扫过——（"脚步一顿"在 ch8 同一章出现两次，纯模板复用）
- **正确写法（片段 ≤150 字）**：
  > 石面在他掌下有了回应，竟透出暖意，顺着小臂缓缓漫上心口——像漂泊多年的人，终于又踏进了家门。（用一句话写碑的回应，不套四字）
  > 陈言停住脚，余光扫过——（一处用"停住脚"，同章另一处用"收住脚"，不重复同形）
  > 他正看得出神，碑周的光影无声荡开，像水面被风拂过。（"忽地一晃"→展开成一句描写）
- **可固化规则（检测用「一」+「名词一动词」）**：
  - **判定边界**：凡"名词+一+动词"四字（或"忽地/猛地/齐齐+一+动词"）用作叙述节拍，无论体内体感（心头一震、体内一热、瞳孔一缩、指尖一颤）还是体外事件（碑身一震、虚影一颤、台心一颤、脚步一顿、脚下一顿、脸一沉、一字一顿、光影一晃、齐齐一颤），一律视为重复率高的廉价写法，要清。
  - **因果线/古纹 motif 的窄例外**：本书核心视觉机制"因果线/纹路"允许用颤动、发亮、一明一灭、一跳一跳等描述，但（a）**同一章内不重复同一四字形式**；（b）优先换动词（颤/亮/沉/绷/浮/荡）而非固守"一X"模板；（c）林小月古纹"一跳一跳"可作 motif，但同章第二次出现须换写法（如"微微起伏"）。
  - **替换方向**：要么给明确外部起因/具体描写（用一句话写反应），要么直接写认知或动作；没有现成四字就展开成一句，**绝不硬套"Noun一Verb"**。
  - **写后自检步骤**：grep 全文 `一(震|颤|顿|沉|晃|响|跳|亮|明|灭|缩|荡|醒|紧|暖|麻|凉|热|烫|疼|松|漾|锐|空)` 与 `猛地一|忽地一|陡地一|蓦地一|齐齐一|心头一|体内一|脊背一|肩背一|手心一|瞳孔一|指尖一|眉心一|碑身一|虚影一|台心一|脚步一|脚下一|脸一沉|一字一顿` ，逐条判是否为"名词一动词"廉价节拍（因果线 motif 按上条窄例外放行并检查章内重复）；同时 grep `了一分` 抓 #020 残留。
  - 与 #020 同源：都是"小体感/小事件突变量词"信号；#020 用"分"做程度量词、本例用"一+动词/四字模板"做突发量词。合并视作同一禁令的两副面孔。

## 案例 #025 — 否定式冗余描写（"不X不Y"打预防针 + 模糊拟人比喻 + "不是"澄清冗余）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-17，ch8 微调：用户原话"不伤人不扑人 像在问他什么 都是老问题了"）
- **现象（用户原话）**：「不伤人不扑人 像在问他什么 都是老问题了」——指 L25「石室四壁浮起几道虚影，不伤人不扑人，只一圈圈绕着陈言转，像在问他什么……这不是攻杀，是传承在验他」中三处同源问题：①"不伤人不扑人"；②"像在问他什么"；③"这不是攻杀"。
- **根因**：①**否定式打预防针**——作者怕读者误会虚影有攻击性，提前用"不X不Y"说它"不做什么"，而非写它正面"在做什么"；读者要的是正面动作，预防针是废话。②**模糊拟人比喻/贴意图标签**——"像在问他什么"给虚影贴了个"在问"的意图，与"像见了自家人""终于踏进家门"同族，用模糊比喻代替具体展示；"在问/在验"的含义应由后文具体事件让读者自己感知。③**否定式澄清冗余**——"这不是攻杀"与"不伤人"重复表达同一意思（虚影无攻击性），前文无攻击暗示时此句纯多余。
- **错误写法（片段 ≤150 字）**：
  > 石室四壁浮起几道虚影，不伤人不扑人，只一圈圈绕着陈言转，像在问他什么。头一道是个后生被指为盗……这不是攻杀，是传承在验他。（否定式冗余 + 模糊拟人 + 否定澄清，三处同源）
- **正确写法（片段 ≤150 字）**：
  > 石室四壁浮起几道虚影，只一圈圈绕着陈言转。头一道是个后生被指为盗……陈言眯眼。这是传承在验他：看不看得清一段缠错的因果。（只写正面"绕着转" + 具体案例展示"验"，不靠否定/比喻打预防针）
- **可固化规则（写后自检排查信号）**：
  - **写"在做什么"，不写"不做什么"**：凡"不X不Y"结构（不伤人不扑人、不攻不守、不悲不喜…）出现在描写里，先判它是不是在"打预防针说某物不做什么"——是则删，直接写正面动作；成语/合理副词状语（不声不响地退下）保留，但"不X不Y+动作对象"式防御性描写一律删。
  - **模糊拟人比喻 = 贴意图标签**：grep `像在[问做看等说]|仿佛在[问做]` 抓"像在问/像在等/像在说"类，改由后文具体展示该意图，不在描写里替物/人下意图结论。与 #007（不贴机制标签）、#022（虚比喻堆叠）同源。
  - **"不是"澄清冗余**：grep `不是[一-龥]{1,4}[，。]` 抓"这不是攻杀/这不是幻觉"式，看前文是否已有正面陈述使该否定多余（无攻击暗示时"不是攻杀"删）。
  - **写后自检步骤**：grep 全文 `不[一-龥]{1,3}不[一-龥]{1,3}`（抓不X不Y，逐条判是否防御性冗余）、`不是[一-龥]{1,4}[，。]`（抓否定澄清）、`像在[问做看等说]`（抓模糊拟人）；三条任一命中即按上列规则清。
  - 与 #007（动作法/不贴标签）、#022（虚比喻）、#024（符号卫生）一致：靠正面具体呈现，不靠否定/比喻/符号打补丁。

## 案例 #039 — 判断/叙述后的「解释性尾巴」（"——X 的那种 Y" 把意思重描一遍）

- **来源**：2026-07-18 ch9 L9。原「陈言瞥了他一眼，便知这人干净——遭了冤、却辩不出的那种干净。」用户指：「——遭了冤、却辩不出的那种干净。 不需要这种多余的解释」。
- **现象**：在已下结论/判断的句子后，再用破折号拖一句"——X 的那种 Y / ——也就是 Z / ——说白了就是 W"把刚说的意思换种说法重描一遍。属 #001 三删三加要删的"复读/拖尾"一类，但更具体：是「判断 + 解释判断」的冗余结构。
- **根因**：AI 下判断后"不放心"，自补一句解释以求明确，实则读者已能从前句+上下文会意，尾巴纯属复读。
- **错误写法（片段 ≤150 字）**：
  > 主角瞥了他一眼，便知这人干净——遭了冤、却辩不出的那种干净。
  （"干净"已立住，破折号后整句是把"干净"换个说法再说一遍）
- **正确写法（判断即止，不补解释；若需补，只补前句未含的新信息）**：
  > 主角瞥了他一眼，便知这人干净。
  （判断落定即收；"遭冤却辩不出"若真要写，应挪到后文借角色反应/情节自然呈现，而非在此复读）
- **可固化规则**：
  - 写完一个判断/结论句后，自问："破折号/逗号后这句，是给了新信息，还是把前句换词重说？"若属后者（"——X 的那种 Y"），删。
  - 解释性尾巴常伪装成"补氛围/补动机"，但若信息已被前句或上下文覆盖，即冗余，删。
  - 真要补充，只补前句未含的**新**动作/新信息（如后文用周野"头回有人替他把清白立住"的反应来呈现"遭冤却辩不出"，而非在判断句里复读）。
  - 自查 grep：判断动词（知/明/懂/看出/断定）后紧跟 `——` 或 `，[^，。]{1,15}的那种` 的多为解释尾巴，逐处砍。

## 案例 #041 — 比喻错配 / 纯装饰俗套比喻（喻体反应与所修饰动作矛盾，或不承载情节信息）

- **来源**：2026-07-18 ch9 L13。原「那银红便随他指尖的触碰微微颤了颤，像被踩了尾巴的蛇。」用户问：「像被踩了尾巴的蛇 蛇被踩了只会颤一颤?」——指出喻体（被踩尾巴的蛇）暗示暴起反扑，与所修饰动作「微微颤了颤」（轻抖）程度/反应矛盾；且"蛇"与上下文（草上被按上去的银红记）毫无关联，是纯装饰俗套。
- **现象**：①比喻本体与喻体的**反应/程度不匹配**——喻体引发强联想（蛇暴起、弓绷断），本体实际只是轻微动作，读者脑中画面与文字打架；②比喻**不承载情节信息**，只是给名词套个俗套意象（像…的蛇/像…的死鱼眼），删掉不影响任何事。两者常叠加：既错配又无用。
- **根因**：AI 写完一个动作后"顺手加个比喻"凑文采，不校验喻体暗示的反应是否等于本体的实际反应，也不问"这比喻有没有多告诉我一点东西"。
- **错误写法（片段 ≤150 字）**：
  > 那银红便随他指尖的触碰微微颤了颤，像被踩了尾巴的蛇。
  > （蛇被踩尾巴读者想到的是咬人/窜起，不是"微微颤"；且"蛇"对"草上银红记"零信息增量）
- **正确写法（比喻要么反应匹配、要么承载信息；否则删掉，只写动作+可读取的结论）**：
  > 那银红便随他指尖的触碰微微颤了颤，竟像是活物。
  > （"活物"反应匹配"轻颤"，且承载信息——这红记不似草木本有、像有物活在其下，直接服务"栽赃"判定；若连这层都不想点，则只留「微微颤了颤」+后文「他心上雪亮」即可）
- **可固化规则**：
  - 写完比喻后自问两句：`喻体暗示的反应 == 本体实际动作吗？` 与 `这比喻多给了读者一点情节/设定信息吗？` 任一为否，删或换。
  - 装饰性俗套比喻（像…的蛇 / 像…的死鱼 / 像…的石头）一律砍，除非它同时揭示设定（如"像活物"揭示红记非自然）。
  - 与 #001（三删三加·删虚词装饰）、#012（机制隐喻·不堆砌）关系：本例是它们落在"比喻"这一具体载体上的子类，专注"喻体-本体反应错配"。

## 案例 #043 — 比喻/描写植入不会兑现的暗示（假伏笔：「像是活物」但它以后不会活）

- **来源**：2026-07-18 ch9 L13。上一轮为治 #041 装饰俗套，把「像被踩了尾巴的蛇」改成「那银红竟像是活物」，自认为"承载信息"。用户追问：「竟像是活物。 以后是不是真的能活? 不能的话这么写的意义是什么」——点破"像是活物"给读者埋了一个不会兑现的暗示（读者会期待红记真活过来），比纯装饰更隐蔽的坑。
- **现象**：为了让描写"承载信息"，用了一个带**未来指向性**的比喻/说法（像是活物 / 仿佛有生命 / 似要挣脱 / 像在呼吸），可故事里该物**永远不会**真的活/挣脱/呼吸。读者把它当伏笔记住，等兑现，等不到就成了烂尾暗示或误导。
- **根因**：AI 混淆了"承载信息"与"制造悬念暗示"——"像是活物"确实比"像蛇"多给了信息，但那信息是**错的方向**（暗示生命/将来会动），而作者真正想说的只是"这红记不自然、非草木本有"。选了个语义外延超出本意的喻体。
- **错误写法（片段 ≤150 字）**：
  > 陈言指尖在盒沿一拂，那银红便随他指尖的触碰微微颤了颤，竟像是活物。
  > （"像是活物"=暗示会活；若红记只是死的因果残痕、永不会活，此暗示不兑现即误导）
- **正确写法（要么去掉误导暗示直接陈述性质，要么删；不留"会活/会动/会挣脱"的假伏笔）**：
  > （删整句。红记"不自然、是被按上去的记"前文已说透，L21「红记纹路是印的残痕」再坐实，无需再用比喻暗示"活")
  > 或直陈性质而不带未来指向：……那银红绝非草木本有之色。
- **可固化规则**：
  - 写比喻/拟态前自问第三问（承接 #041 的两问）：`这个喻体/说法有没有暗示某种"将来会发生"的性质（活/动/挣脱/呼吸/苏醒）？该性质以后会兑现吗？` 不会兑现 → 换成无未来指向的直陈，或删。
  - 尤其警惕"像是活物 / 仿佛有生命 / 似要挣脱 / 像在呼吸 / 隐隐搏动"这类给死物注入"将活"暗示的说法——除非该物设定上真会活。
  - 与 #041 关系：#041 治"喻体反应错配/纯装饰"，本例治"喻体虽承载信息但方向错、埋不兑现的假伏笔"，是同一自问链的第三道闸。

## 案例 #046 — 疯狂点题 / 强行串线：作者借角色反应把小事往大阴谋或主题上绑（无证据、无必要）

- **来源**：2026-07-19 ch9 L19（两版均犯）。第一版：「玄阴宗、灵草、拿门规压一个散修……他心里那半句残文又浮上来。钱万三那本对不上的旧账里，被裁决司旧印改过的裁断……若周野这案也牵着那枚印，那便不是一桩盗草的小案。」用户：「这就直接看出来了?」第二版（自以为修好）：「这形状，他不是头回见。钱万三那本对不上的旧账里……眼前这桩听着也是拿规矩压人的勾当，可到底压没压……识海里那半句残文动了动，没浮上来，又沉了下去。」用户：「这不还是同样的问题吗? 怎么就能看出来跟以前的旧帐相同了? 都还没接呢, 你这个疯狂点题的毛病什么时候能改? 写了那么多技能不管用?」——两版本质同一病：还没接案、连人没见草没摸，就借主角内心把眼前这事往大阴谋（旧印/旧账/残文）上绑。
- **现象**：作者有「每场戏都要朝主线/大阴谋点头」的强迫症——角色刚听到一件小事，内心独白就自动调取旧账、残文、预言、主题，做出「这案不简单/牵着那枚印/同形状」的提前定性。读者视角：主角像未卜先知，且每个小案都被剧透成大局一环，悬疑感与角色可信度双崩。
- **根因（为什么是假修）**：把「主角能感知大阴谋」写成**结论或模式识别**，而非「该去查的疑点」。区分点：①有证据/已发生 → 可认（如 L45 摸着草根银红、看纹路才认出旧印，是挣来的）；②无证据/未发生 → 任何「认出同形状/牵着那枚印」都是作者替主角知道了他无从知道的事 = 点题。上一轮把「看穿结论」软写成「认出同形状」，仍是同一越权，只是换了遮羞词。
- **错误写法（未接案就绑）**：
  > 陈言没立刻应。玄阴宗、灵草、拿门规压一个散修……这形状，他不是头回见。钱万三那本对不上的旧账里，被裁决司旧印改过的裁断……若周野这案也牵着那枚印，那便不是一桩盗草的小案。
  > （人没见、草没摸，已「看出」牵着旧印——既未卜先知，又剧透 L45）
- **正确写法（只写眼前，大线留到挣来的地方）**：
  > 陈言没立刻应。玄阴宗是东域数得着的门派，为三株草就要废一个散修、把他逐出东域。眼前这人跪得膝盖渗血，求的不过是让他去台前看一眼。
  > （只评估眼前利害与人的处境；旧账/旧印/残文一律不出现。大阴谋线在 L45 靠物证「银红残痕=旧印」才落地，夜思段在查完才收线——都建立在「已发生/有证据」上）
- **可固化规则**：
  - 写角色反应/内心独白前自问三问：`①这角色此刻掌握了什么证据/信息？②他现在下的判断，有没有超出这些证据能支持的范围？③这句是不是在替作者朝主线/主题「点头」（点题）？` 任一问答「超了/是在点头」→ 删，或改为「需查证的疑点」而非「已知结论」。
  - **大阴谋/主题线，只在「已发生且有证据」处落地**（如亲眼见物证、已翻过案、已对账）。绝不在「刚听说/还没接/还没查」处提前定性。
  - 残文/预言/旧账/传承这类「挂着的线」，在前期只能作「未浮起的警觉/直觉」，不能作「已确认的连接」。写「动了动，没浮上来」可以；写「牵着那枚印/同形状」不行。
  - 变体警示：①本地角色台词里报只有对读者才需交代的地名/背景（如本地人台词说「落霞镇有位陈讼师」）——也是作者借角色之口替读者报幕，同属本例根因；②任何「这案不简单/水很深/牵出某人」式无证据提前定性。
  - 与 #013（前世越界）、#002（设定一致·不杜撰归因）、#045（爽感不自夸）、#017（点题拖尾）关系：#013 治「用前世证明无优势的事」；#002 治「乱归因」；#045 治「当场解说谋算」；#017 治「章尾重述主题」。本例治「未卜先知的提前串线/点题」，是它们的上游根因之一——写作中先过本例，再检查那几条。

## 案例 #047 — 描写失真与俗套：时间/物理尺度夸张 + 套路化情绪动作/比喻

- **来源**：2026-07-19 ch9 L31、L29（同一章连续两处，用户当场点破）。L31 原：「他二十出头，下巴尖削，眼窝深陷，是长久饿出来的憔悴。一夜囚笼把他熬得脱了形，抖得像风里的叶子，眼神空落落的……」用户：「抖了一夜?那不是早该死了?」L29 原：「人群里有个瘸腿的老散修，攥着拳，指节发白，像是想起了自家什么旧事」用户：「攥着拳，指节发白 这个太套路了」。
- **现象**：两类常见失真——
  1. **时间/物理尺度夸张（逻辑崩）**：把需要长年/重创才有的状态，压缩到极短时间或轻微诱因上。如「一夜囚笼→脱了形」——一宿关笼最多让人疲惫灰败，不可能「脱了形」（那是经年穷饿/久病才有的外貌改变，且前文 L31 自己已说憔悴来自「长久饿」）。再细读「一夜…抖」会被读成「抖了一整夜」，物理上荒谬（持续颤抖整夜足以虚脱致死），读者自然反问「那不是早该死了?」。
  2. **俗套情绪动作/比喻（套路化）**：用网文万能模板写情绪，脱离角色具体处境。「攥着拳，指节发白」= 写「憋怒」的通用动作，落到谁身上都行，浪费了角色最特异的身体特征（那瘸腿老散修的记号是被门规废了的腿）；「抖得像风里的叶子」= 写「害怕/虚弱」的通用比喻，无新意也无信息量。
- **根因**：作者图省事，用「读者一看就懂的现成套话」代替「从这角色、这处境长出来的具体描写」。结果：①尺度夸张会触发读者物理常识警觉（"一夜怎么脱形/抖一夜不死?"），直接出戏；②套路动作/比喻零信息量，还掩盖了本可用来塑造角色的细节。
- **错误写法（失真/俗套）**：
  > 一夜囚笼把他熬得脱了形，抖得像风里的叶子。（一宿→脱形，尺度崩；俗套比喻）
  > 台上跪着个瘦长年轻人，下巴尖削，眼窝深陷，是长久饿出来的憔悴，本就单薄。（「瘦长/尖削/深陷/憔悴/单薄」五个近义形容词全在说「瘦弱」一件事= 同义堆砌水字数，零信息增量，读者只觉被形容词轰炸，没多知道角色一分）
  > 瘸腿老散修攥着拳，指节发白，像是想起自家旧事。（通用憋怒模板，浪费其废腿特征，且直说「想起旧事」= tell 非 show）
  > 陈言指尖搭上银红……果然又是它。他心里雪亮，面上却不动声色，把盒还了回去。（「果然」= 得意式已知定性，边缘踩 #046；「心里雪亮/不动声色」= 深藏不露主角俗套心境描写，tell 非 show，靠装深沉代替具体反应）
  > 台下静了一瞬，随即炸开。请天道裁，那是要天道显像、当众验因果的。心里没鬼的人才敢立那个誓。众人的眼睛齐刷刷望向台上那两个弟子，有人替周野捏了把汗，有人已在低声骂玄阴宗欺人。（机制复读：核心设定 ch1-5 已建立，ch9 又向读者重讲一遍= #007/#017 类设定复读；「心里没鬼」= 作者跳出来替读者总结的俗套大白话；「有人…有人…」泛群像排比= 用概括代替具体场景，且与紧接的具体弟子反应撞功能，啰嗦）
  > 两个弟子对视一眼……台下的人都看得分明：这草有鬼。那瘸腿老散修……浑浊的眼里第一回有了点热气。周遭的议论声也压不住了……往后谁再摊上这号冤事，也知道该往哪家门上寻人了。（「看得分明这草有鬼」= 无证据替读者下具体结论+朝栽赃点题，踩 #046 边缘；「眼里第一回有了点热气」= 写被触动的俗套情感，且 L29 已用「按废腿+盯周野」含蓄 setup 让读者自读，此处又直白念破= tell 毁 show 留白；「往后冤事找陈讼师」= 配角变作者嘴替，无情节支撑预告主角声望，踩 #046）
- **正确写法（尺度守真 + 从角色长出）**：
  > 他二十出头，长久穷饿熬得身形单薄。昨夜关在笼里没敢合眼，这会儿脸色灰败，被镣铐勒着，身子禁不住发颤，眼神空落落的……（只点一次「单薄」；瘦弱全靠「脸色灰败/镣铐勒着发颤/眼神空落落」这些具体状态演出，不靠「瘦长/尖削/深陷/憔悴/单薄」一堆近义形容词轰炸——那叫同义堆砌水字数，零信息增量）
  > 瘸腿老散修手按着自己那条废了多年的腿，目光却牢牢黏在台上那跪着的年轻人身上。（动作从其身体特征长出；不直说「想起旧事」，用「按废腿+盯周野」让读者自读「他看见当年的自己」= show don't tell）
- **可固化规则**：
  - 写状态/外貌前自问：`这个变化需要的「因」（时间/创伤/境遇）和「果」匹配吗？` 把「长年/重创才有的状态」套到「一夕/轻微诱因」上 = 尺度失真，立刻改：要么拉长成因时间，要么只写该时长能有的后果（一夜→疲惫/没合眼/灰败，不→脱形/白发/濒死）。
  - 写情绪动作/比喻前自问：`这个动作/比喻换成别的角色也成立吗？` 若「删掉角色名也读得通」= 套路，须换成本角色**特异处境/身体特征**长出的动作（瘸腿→按废腿；负债→摸空钱袋；旧伤→按旧伤处）。
  - 俗套比喻黑名单（见到就换）：「抖得像风里的叶子 / 脸色白得像纸 / 攥紧拳头指节发白 / 像被踩了尾巴的猫 / 眼里迸出光（滥用时）」——不是绝对禁用，而是「有更贴本角色的具体写法时优先具体」。
  - 俗套心境/城府黑名单（见到就换）：「心里雪亮，面上不动声色 / 不动声色 / 果然又是它 / 果然是它 / 一切尽在掌握 / 成竹在胸 / 心里没鬼（的人才敢…）」——看穿真相/局势后的反应要落**具体动作或具体理由**（为何不当场点破、下一步查什么），不靠「装深沉套话」盖过去。「果然」类尤忌：它暗示主角早知道，边缘踩 #046 的「提前定性」，即便此刻已有证据也要改成中性比对（「对得上」「纹路缺角认得」），不用 smug 口吻。「心里没鬼」类总结句= 作者替读者把显而易见的事再念一遍，删掉，让读者自己推。
  - 群像反应不写「有人…有人…」泛排比：读者要的是**一两个具体的人/动作**，不是概括清单。写看客反应前自问：`我有没有一个具体的脸/手/动作可以代替「众人」「有人」？` 若已有紧接的具体人物反应（如某角色的具体神态动作），泛群像那句直接删，避免功能撞车啰嗦。机制/设定已在前文（尤其前几章豁免期）建立过的，后文**不再向读者重讲**（那是 #007 机制裸奔 / #017 点题拖尾的复读变体），用「角色当下的具体动作 + 一句带过戏剧张力」承接即可。
  - show don't tell：情绪不靠「像是想起旧事 / 心中一阵酸楚」直说，靠具体动作+读者自读。已用具体动作/细节做了含蓄 setup（如 L29「按废腿+盯周野」让读者自读「他看见当年的自己」），后文**绝不直白念破**——再写「早年他也这样被废了腿 / 眼里有了热气」= tell 毁 show 留白，且「眼里有了热气 / 眼里迸出光」本身就是写被触动的俗套情感。配角/路人不是作者嘴替：不为预告主角声望/地位说工具话（「往后冤事都找陈讼师」类），那是无情节支撑的提前定性，踩 #046。
  - 反差/幽默桥段（会心一笑）要短、靠错位、不解释：喜剧效果来自「读者预期」与「实际落点」的错位（如悲情磕头时刻主角先心疼地砖），不是作者把笑点掰开讲。写成大段解释（他多心疼/补砖多麻烦/为什么好笑）反而稀释幽默、显啰嗦。原则：一句到位，show 错位，让读者自己笑；忌在笑点后追加说明文字。笑点句里也不重复已建立的场景标签（开场已点明「事务所」，笑点句不必再提）。
  - 写外貌/状态时自问：`这几个形容词说的是不是同一件事？` 若「瘦长/尖削/深陷/憔悴/单薄」都指向「瘦弱」= **同义形容词堆砌水字数**，立刻砍到只留一处，其余用**具体可感的状态/动作**代替（脸色灰败、镣铐勒着发颤、眼神空落落、手腕在镣铐里显空荡）。真要显瘦靠具体画面（风一吹衣摆贴住肋骨、镣铐圈出一圈空荡），不靠形容词表轰炸——堆砌零信息增量，读者一眼识破「在凑字数」。
  - 介绍物品 / 灵草 / 法宝 / 功法 / 阵法等设定时，须给**具体可感的物性**，不可「贴标签 + 补一句很值钱」敷衍了事。动笔前自问：`读者看完能"看见/摸到/闻到"这东西吗？` 笼统写法如「玄霜草，东域出了名的阴寒灵草，一株抵散修数年资粮，多拿来凝丹吊性命」只抛了个抽象结论「很值钱」，读者根本没见到草本身，读着像设定表抄录、敷衍。正确做法：细节从角色**当场观察**长出——长相（叶比柳叶还窄、边缘细密锯齿）、那层霜白究竟是什么（自身泌出的寒脂、非夜露、离土不化）、几年长成（三五年才一寸高）、长在何处（背阴峭壁缝，东域数得着的崖场归某宗看管）、炼出作什么用（刮霜白炼丹、压火毒续性命）、对谁意味着什么损失（足年好货、三株同失是实打实损失）。用感官动词落地（指腹搓开是凉的、带薄荷混铁锈清气、掐断渗淡汁），不靠形容词表轰炸；价值由物性本身显出，不必另写「很值钱 / 抵数年资粮」一类空泛估值句。让设定「活」在角色的眼手口鼻里，而非作者的说明文字里。
  - **收尾/升华段忌「借角色口吻一口气甩全书主题宣言」**：章节末尾主角忽然念出一段普世公理式总结（如「最可怕的不是明着的冤，是连受害者自己都信了的冤」+「要么把那枚印从根上拔了，要么让天下人都长出看得破因果的眼」），既是作者跳出来替剧情做交代（tell 非 show），又常与后文情节**重复**——后文本会用具体行动落地同一主题（如教配角「替我多一双眼睛」、把案并成摊到天下人眼前的状）。修法：升华不从空中掉公理，而从**当场动作/具体处境**长出（笔尖悬着、看着眼前这人，让领悟随描朱砂的手停顿浮出）；同一主题只许在**一处**用情节呈现，收尾段若与后文撞主题，删收尾的宣言、留后文的呈现。grep 自查：收尾段若出现「最可怕的是…/说到底…/要断了这祸根，要么…要么…」类宣言句式，先查后文是否已有同主题的具体情节，有则删此处。
  - **章节中段忌「宿命感慨式内心独白」（命运自我陈述）**：角色忽然跳出当前动作做一段宿命/传承式内心戏（如「自己这看破因果的本事，从落生起就是冲着这只手来的」「初代讼师留规矩，怕也是为着这一日」「他这道传承，算是接了初代讼师的班」），是作者跳出来替角色做命运总结，纯 tell、零当场信息，且常一口气连写两三句同质感慨叠在一起显啰嗦。修法：①凡「从落生起/生来就是/为着这一日/接了某人的班/注定」类宿命句式，一律删；②若其中夹着**真实机制/设定推进**（如「镇器无主，那套改因果的规矩便烂了，底下人学着用」），只留该机制句、砍光感慨外壳；③宿命感靠后文情节呈现（角色一次次被卷入、被盯上），不在中段用内心戏念出来。与 #014（少写内心）同源：内心戏只许长在「当下具体处境推出来的下一步」上，不长在「宏大命运」上。
  - **「渲染分量」与「给出的数字/结果」须成比例（忌前重后轻）**：前面用铺陈把分量撑得很重（如列举多个案例名「玄霜草、通源号商契、沈舟血案……桩桩件件都缠着同一枚印」、感慨「这枚印平日改过多少无辜人的果」），后面给的定量结果却很小（如「七桩有余」），读者会觉「前面渲染那么多，就这？」比例失调、分量落空。修法：①若铺了多案例/强分量，给出的数字须匹配量级（用模糊词「数十桩/几十桩/不止一二十桩」等，仍遵守模糊数字规则、不写精确数）；②若确只想写「小范围」，则前面铺陈须收力，不可又列多名案例又给小数字；③可加限定句化解「只这些」的违和感（如「这还只是 X 旧账里一眼能辨出印痕的，经年陈档不计其内」），让小数字成为「冰山一角」而非「全貌」，并顺接后文「别处还改过多少」。本质是**铺垫的体量与回报要等量**，属 #046/#047 家族（失真/比例失调）。
  - **对白/台词里的「抽象比喻·双关」须给读者可落地的解读（忌玄虚飘忽、读者看不懂）**：角色台词用了没落地的双关或隐喻（如「往后你认字，不光认人间的字，也认这天道认下的因果」），读者既不知这比喻在此刻指什么**具体动作**，又接不上为何突然说这句，等于「看不懂」；若再把主角**专属能力**（看破因果）用「你也认因果」的方式分出去，还造成设定含糊。修法：双关须先有一个读者熟悉的具体锚点（如「认字念书，也不光认书上的字」），再顺势转到**可操作的具体行为**（「镇上谁又平白摊了冤、谁像叫人借了身子去演一出戏，你替我多留一份心」），让抽象词（认因果）变成读者能看见的动作；避免用「认 X 之道 / 你也看破 X / 懂这天道」类句式把专属能力分出去。与「介绍物品须具体物性」「去旁白」同源：凡台词出现「认 X / 懂 X / 看破 X」类抽象授业句式，先问读者能否据此想象出一个具体动作，不能则落地。
  - 与 #046 关系：#046 治「无证据提前定性（逻辑越权）」；本例治「有证据但描写尺度/用词失真（物理越权）」。两者都是「作者偷懒用套话越过真实」，常在同一段并发，写完过 #046 三问后顺手查本例。

## 案例 #053 — 讲故式旁白（Tell→Show 错误示范史，正解见 #054）

- **来源**：2026-07-21《告天》ch10 周户籍官段（详见 #054 来源）。
- **现象 / 根因（两次错解迭代）**：
  1. 初版把背景/往事当**旁白信息摘要**喂（「他坐了数十年/见过太多权压理的场面/自那以后学会了低头」）——Tell 重 Show 轻，读者像听 narrator 报履历，没被拉进角色此刻的身体与记忆。
  2. 改成具身画面后，摄像机仍架在**配角脑内**（报道配角自己都未必说出口的往事）——主角限知视角下仍是旁白腔，用户实测反馈「还是第三视角、仍在讲故事」。
- **正解**：两个问题同根——「没先定摄像机该架在谁身上」+「想法/历史/主角推测禁入旁白」。真正的分层正解见 **#054**，本条仅留作错误示范史，勿再独立当解法用。

## 案例 #058 — 假转折（「可/但」硬连无因果的两事）

- **来源**：2026-07-21《告天》ch10。林小月赢官司后：「先生你本事是大，可镇上那家炊饼也小气得很，饼越做越小，坑得我好苦。」用户：「这句内部因果不对」。
- **现象**：用「可/但/然而」连接两件本无转折/因果关系的陈述，转折词落空，句子逻辑断裂；读者读到硬拗的转折会出戏。
- **根因**：作者为凑「先扬后抑/先夸后损」的节奏，强行给无关两事套转折壳；或前半句已表过情绪，后半句又拐去夸/损别处，节外生枝。
- **错误写法**：
  > 先生你本事是大，可镇上那家炊饼也小气得很。（「本事大」与「炊饼小气」无转折关系，硬连）
- **正确写法**：
  > 都怪镇上那炊饼铺忒抠，饼越做越小，一个下肚跟没吃似的，早晓得多啃两口硬饼垫垫了。（碎嘴直接承接「饿」，因果自洽）
- **可固化规则**：
  - 下转折词前自问：「前后两句真有转折/对立关系吗？」无 → 删转折词，或换承接/因果连接（「……所以……」「……；」）。
  - 同一吐槽内立场须自洽：别前半句说「不该吃硬饼」、后半句说「该多吃硬饼」自反（与 #028 内部属性矛盾、#012 逻辑错同查）。

## 案例 #059 — 同义堆叠啰嗦（让步理由不对 / 重复铺陈来头）

- **来源**：2026-07-21《告天》ch10。陈言劝林小月瞒古纹：「这纹连着的东西，未必是林家担得起的。」林小月：「那便瞒。横竖它又不疼不痒，我自个儿也只当是块胎记，省得我瞎操心。」用户：「这句也有点奇怪，跟上下文穿起来感觉啰嗦」。
- **现象**：① 用 2-3 个同义词叠说同一层意思（「不疼不痒/只当胎记/省得瞎操心」）；② 让步/解释理由与角色真实动机错位（她真会让步是因信先生，不是因「纹不重要」）；③ 把已写过的来头/神秘感再平铺一遍（「连着上古/连着很远/连着一处没资格碰的源头」）。
- **根因**：作者图「语气饱满」硬凑三层同义，或没接上前文已建立的动机，用错位的理由填空。
- **错误写法**：
  > 那便瞒。横竖它又不疼不痒，我自个儿也只当是块胎记，省得我瞎操心。（三层同义 + 理由错位）
- **正确写法**：
  > ……行吧，先生说的，总没错。（因信先生让步，一句收口，不堆同义）
- **可固化规则**：
  - 同一层意思只留一句；「不疼不痒/只当胎记/省得操心」式同义堆叠属 #001 三删三加「删复读」的下位，写时自删。
  - 角色让步/解释的理由须与其**真实动机**一致（信先生→因信让步；不是因「事不重要」），理由错位即删。
  - 来头/神秘感前文已写过的，不在后续角色内心/对话里再平铺（与 #017 章尾不总结、#001 删复读联动）。

## 案例 #064 — 长篇小说忌过度修辞 / 指代失当 / 无信息量总结（ch11 多轮精修提炼）

- **来源**：2026-07-22《告天》ch11 用户连续十余轮精修反馈（赵无极出场段 L63–69、认罪段 L89–103）。用户原话：「过度的修辞在长篇小说中会让人烦躁，会感觉浮夸」「那桩压了多年还是感觉浮夸」。
- **现象（六类，均来自真实反馈）**：
  1. **过度修辞 / 浮夸**：出场句用虚比喻收尾（「指腹摩挲墨迹，像要把这话，带回宗门去问」——比喻不搭且"问"悬空未兑现）；内心独白叠比喻（「宗门蒙尘」「缝隙里洗不净的灰」）；收尾句文艺腔（「目光像一片雪落在后颈，凉而轻」「笔直通向不可知的高处」）；轻抒情修饰（「压了多年的旧案」）。
  2. **指代 / 概念失当**：冒号错挂（「他第一次疑上那句教诲：若感应会错……」——冒号后实为角色自己的质疑，非教诲内容，且前文无具体训词可指回）；量词错配（「一句神识感应」——「一句」是话语量词套术法名词，读如半句话）；刻意量词（「那桩旧事」太刻意）；视角称呼矛盾（同段先写「陈言」又突然称「这散修」，赵无极与主角早相识，生分且视角打架）。
  3. **角色认知越界 / 未卜先知**：角色替作者断言清白（「那弟子的冤，翻得了」「怕当年那桩，本就断错了」——读者问"他怎么知道有冤"）；且「翻得了」与「冤」同义，摘字不摘逻辑。
  4. **无信息量内心总结 / 啰嗦复述**：把已坐实的事再复述一遍（「两卷假契，落款金纹收势偏急，分明同一只手……那手不亲自出面，借残部当棋子，躲在上头背后」——周虎同手法前文已坐实，此处零增量）；串线感慨「若白纸黑字都能仿出壳，东域的理从哪寻」无新信息。
  5. **语气 / 肢体越界**：该调侃却怒斥（对方拿「天道为证」压人，主角应把原话掷回脸上调侃，却写「天道反噬，你担得起？」怒斥，且「你担得起＋立不立这道誓现在就能验」翻来覆去）；认罪肢体无厘头（前一刻冷笑、被打脸后「腿软扑通跪」——既秒跪不合人设，又「跪」操作莫名其妙，且与后文「刚要出口被咽回」矛盾）。
  6. **编辑连锁漏改**：换案子主体（赤霞谷残部↔青松门 → 乌沙宗残部↔栖梧宗）只改前半，后文「赤霞谷残部」「灵田归青松门」漏改留矛盾；改「重验」却前文无「验」概念铺垫（「又如何经得起重验」突兀）。
- **根因**：AI 长文生成默认用「文艺修辞＋角色内心剖白＋作者总结串线」填充篇幅与"显得有深度"，而非用动作/事实推进；且多轮局部 Edit 改前半忘后文，产生连锁漏改。长篇小说里这类"显得用力"的写法会累积成阅读疲劳。
- **错误写法（片段）**：
  > 指腹摩挲着墨迹，像要把这话，带回宗门去问。（虚比喻收尾，"问"悬空）
  > 他第一次疑上那句教诲：若感应会错……（冒号错挂质疑为教诲）
  > 那弟子的冤，翻得了。（角色断言清白，越界）
  > 陈言把这一幕与周虎那桩并在一处。两卷假契，同一只手……那手不亲自出面……（零增量复述）
- **正确写法（片段）**：
  > 合上册子，指节按了按硬壳。（务实动作收尾）
  > 自幼被教以「神识辨善恶」……他第一次对这套信条生了疑：若感应会错……（坐实教诲原话＋冒号明确为疑）
  > 这般定下的罪，若也请天道当场为证，还立得住么？（只质疑依据，不断言清白）
  > 陈言把刀客咽回去的那半句「上头」，暗暗记下了。（只保关键伏笔，不复述已坐实信息）
- **可固化规则**：
  - **删过度修辞 / 浮夸**（长篇小说最高频坑）：角色内心独白、章节收尾句、出场句禁用文艺比喻与虚词堆叠；能用平实动作或对话传达的不堆比喻。轻修饰（「压了多年」「那桩」刻意量词）也属浮夸，一并去。自查：凡「像/仿佛/如＋抽象喻体」作收尾或内心、凡「不可知/说不清/压了X年」式抒情，必查能否换成动作/事实。
  - **删无信息量总结 / 复述**：已在前文坐实的信息（某手法相同、某伏笔已埋），禁止后续用角色内心/作者旁白再完整复述。角色内心只承载**本段新推到的下一步**（如"记下这半句上头"），不重复旧知。自查：凡「X 把这一幕与 Y 并在一处」「同一只手/同一手法」类串线总结，回查前文是否已写过，写过即删。
  - **概念 / 指代须自洽**：冒号前后同主语（别把"角色质疑"挂成"教诲内容"）；量词匹配名词（「一句」只接话语，术法用「单凭/一道」）；同一视角段落称呼统一（早相识角色叫惯称，不突然"这散修"）；改词后 grep 全段清零旧写法。
  - **角色认知不越界**：角色只能质疑"依据/过程"，不能替作者断言"结论/清白/能翻案"。凡「冤/翻得了/本就断错」改回"对依据存疑"的程序性表述（呼应前文已建立机制，如"请天道为证还立得住么"）。与 #029（认知越界）、#038（未卜先知）同根。
  - **语气贴人设、肢体有理由**：对方拿"天道为证"压你，掷回原话是调侃/激将而非怒斥；认罪须有缓冲（先想硬撑→被证据压垮），不秒跪；肢体动作须有情境理由（认怂用"肩膀一垮"而非无厘头"扑通跪"）。与 #044（态度≠病句）、#050（trait 落地）互补。
  - **编辑连锁纪律（派生）**：凡局部改写涉及"换名/换宗门/换案由"，必 grep 本章全文清旧写法，尤其后文引用（"归 X 门""X 残部""上头指向"）——只改前半漏后半是高频 bug。
