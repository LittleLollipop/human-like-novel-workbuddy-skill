# 错误案例 · 语言/语病

## 案例 #005 — 看不懂的因果句（词义模糊 + 因果不清）

- **来源**：用户长篇连载项目反馈（中段章节）
- **现象**：「今年某宗门下脉三个弟子卡在炼气瓶颈，半数是灵气不足所致」——「下脉」词义不明，整句因果读者读不通。
- **根因**：用了作品内部黑话（下脉）且未交代因果链条。
- **错误写法**：「今年某宗门下脉三个弟子卡在炼气瓶颈，半数是灵气不足所致。」
- **正确写法**：「今年某宗门下游一脉，有三个弟子卡在炼气瓶颈上——其中两个，分明是灵气亏虚拖的，寻常年景哪会这样。」
- **可固化规则**：避免作品内部黑话；因果句须把「因→果」说清，模糊处加一句解释或落到具体现象。

## 案例 #010 — 台词未查语用限制（人称/谦敬绑定、语体、视角）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-13）
- **现象**：把描写/成语直接搬进角色台词，忽略词语的**使用条件**。例：称某角色台词「化用刻骨铭心」，但「刻骨铭心」是带语体（庄重书面）+ 视角（事后回顾）限制的描述词，让角色**当面、即时**对师长说会立刻"隔"；又如谦敬辞「犬子」只能称己方孩子，绝不可称对方孩子——人称绑定是绝对限制，违反即笑话。
- **根因**：只做修辞归类（像不像、美不美），漏了**语用层**（能不能、该谁说）。修辞管表达效果，语用管使用条件；写台词先过语用，再过修辞。
- **错误写法（片段）**：
  > "您教我的问心二字，我刻在骨头上。"（把"刻骨铭心"拆成生造口语当面说，既不满足"化用需有可辨识源文本"的前提，又撞上原成语的语体/视角限制）
- **正确写法（片段）**：
  > "您教我的问心，我从未敢忘。"（第一人称、当面、即时、对师长——无任何语用冲突，自然语言）
- **可固化规则**：
  - 写台词下笔前查三道语用限制：①**人称/谦敬绑定**（犬子/令郎/家父/尊夫人等绝对限制，谁说、对谁说焊死在词里）②**语体绑定**（书面庄重 vs 口语随意，场合错配即隔）③**视角/时态绑定**（回顾性、旁观性描述词不宜当下当面说）。
  - 修辞是加分项，语用是及格线；语用不过关，修辞再漂亮也不能用。

## 案例 #024 — 标点符号滥用（破折号/省略号/强调引号堆叠，AI 写作典型痕迹）

- **来源**：用户长篇连载项目反馈（修真+规则辩论题材，2026-07-17，ch8 九修：用户原话"ch8还有大量的符号乱用——太多了"）
- **现象（用户原话）**：「ch8还有大量的符号乱用 —— 太多了」——指破折号（——）在 3600 字体量里出现 22 处（约每 150 字一个），另有多余省略号、以及把「」当强调/概念引号反复堆（"对不上"×3、"验"、"问"、"枷锁"）。
- **根因**：作者把破折号当万能连接符——appositive/aside/转折/比喻引子全用「——」；把省略号当犹疑/停顿填充；把「」当 air-quote 强调普通词。三者叠加 = 全文符号密度过高，是 AI 生成文最显眼的特征之一，读者一眼出戏。
- **错误写法（片段 ≤150 字）**：
  > 陈言看得出神——壁上刻的桩桩件件，都是讼师为人立誓、辩冤断案的旧事。（转折/解说硬套破折号）
  > 钱万三那本「对不上」的账册……东家疑那些「对不上」的裁……碑会「问」人。（同一概念词被「」标 3 次 + 省略号滥用）
- **正确写法（片段 ≤150 字）**：
  > 陈言看得出神，壁上刻的桩桩件件，都是讼师为人立誓、辩冤断案的旧事。（逗号/句号替代破折号）
  > 钱万三那本对不上的账册……东家疑那些对不上的裁……碑会问人。（去掉强调引号作普通短语；省略号仅留对话自然断续）
- **可固化规则（写后自检硬性指标）**：
  - **破折号（——）**：单章（3500–4500 字）出现 **≤2–3 处** 为正常；超出一律改逗号或句号。仅两种情形可留：①引号内表示**物理文本被截断**（如碑文刻到一半劈断「天道公正，实为枷锁——」）；②**真正的突然打断/话题硬转**且逗号句号确实不通。凡"解说/举例/比喻引子（像…）/并列说明"前的「——」一律改逗号。
  - **省略号（……）**：仅留**对话中角色自然犹疑/声音断续**、或**神秘声音/传达被掐断**处；叙述性省略号（"他沉默了……""往事如烟……"）一律删，用动作/环境承载。不堆。
  - **强调/概念引号（「」当 air-quote）**：普通词/概念（对不上、验、问、枷锁、锁…）**不用「」标**，作普通行文；「」只用于①人物原话对话；②引述碑文/手书/文献原文；③书名/篇名。同一概念词在章内被「」标 ≥2 次即违规。
  - **写后自检步骤**：grep 全文 `——` 数个数（>3 必改）；grep `……` 逐条判是否在对话/声音；grep `「[^」]{1,4}」` 抽查是否为强调引号（非对话/引文/书名）。
  - 与 #006（符号过度复读）关系：#006 治"同一符号在文中反复复读同一信息"，本例治"符号种类本身滥用/密度过高"，是更底层的标点卫生。

## 案例 #034 — 长串「的」定语导致归属歧义（结构语病）

- **来源**：2026-07-17 ch9 L1「原是玄阴宗沈舟案的师兄」被用户指「这句有语病」；查源头在 ch4 L97 同源句。
- **现象**：「X的Y案的师兄」这类多重「的」叠加，读者会把「X的Y案」读成一个整体（X 的 Y 案），后接「的师兄」→「Y案的师兄」，到底谁是谁的师兄含糊不清。
- **根因**：AI 写背景交代时爱用长串「的」串成一句、省略标点断开，导致定语中心语归属歧义。属纯语法语病，非设定问题。
- **错误写法（片段 ≤150 字）**：
  > 执印人叫墨通，原是玄阴宗沈舟案的师兄。
- **正确写法（片段 ≤150 字）**：
  > 执印人叫墨通。墨通原是玄阴宗弟子，也是沈舟的师兄。
  （用句号/逗号断开门派身份与同门辈分；或重组为「玄阴宗弟子墨通，乃沈舟师兄」）
- **可固化规则**：写含多重「的」的定语串（A的B的C、X的Y案的Z）时，写完通读一遍——若去掉中间名词后「的Z」会挂错中心语，即说明定语歧义，用逗号/句号断开或重组为「A的B，也是C的D」。可用 grep 自查：`[一-龥]+的[一-龥]+的[一-龥]+的` 长链逐一排查归属。

## 案例 #037 — 无着落被动 / 与已写动作重复矛盾（"被X"施动者未建立，或重描刚写过的动作）

- **来源**：2026-07-18 ch9 L5。原「进门先跪，脑门磕在瘸腿凳上，咚一声……被按着跪下时，他抬头望向陈言」。前文已写"进门先跪"（跪完、磕了凳），后文又"被按着跪下时"——既与"先跪"时序冲突（跪都跪完了何来"跪下时"），又凭空冒出"被按着"的施动者，前后文并未写谁按他。用户指：「被按着跪下时 ???被按着?」。
- **现象**：①用"被X/被人X"被动句式，但 X 的施动者在当前场景前文并未建立（读者不知谁在按/拽/推）；②同一动作在前文已叙述过，后文又用被动重描一遍，造成时序矛盾或重复。
- **根因**：AI 习惯用"被按着/被拽着"等被动填充动作细节，却未先交代施动者；也常在前句已写完某动作后，后句再以"X时"重描同一动作，忽略时序自洽。
- **错误写法（片段 ≤150 字）**：
  > 进门先跪，脑门磕在瘸腿凳上，咚一声。他约莫二十出头，眼窝深陷。被按着跪下时，他抬头望向主角，喉头动了动……
  （"先跪"已发生，又来"被按着跪下时"；"按着"的施动者未建立）
- **正确写法（与已写动作接顺，不重复、不冒施动者）**：
  > 进门先跪，脑门磕在瘸腿凳上，咚一声。他约莫二十出头，眼窝深陷。他跪着抬头望向主角，喉头动了动，像是攒着话又不敢开口。
  （用"跪着"承接"先跪"，无额外被动、无矛盾）
- **可固化规则**：
  - 写"被X"前，确认本场景前文已出现 X 的施动者（谁按/拽/推）；未建立则改用主动或换表述，或先补"两个护卫按他跪下"之类再接被动。
  - 前句已写完某动作（跪/进门/落座），后句勿以"X时/正X着"重描同一动作造成时序矛盾；用状态承接（跪着/站着/坐着）而非重述动作。
  - 自查 grep：`被[一-龥]{1,3}着`、`被[一-龥]+时` 逐处确认施动者已建立、且与上下文动作不重复。

## 案例 #040 — 多候选里用裸称指代（"弟子/青年/那人" among 多个同类，读者不知指哪个）

- **来源**：2026-07-18 ch9。前文「两个玄阴宗弟子押着个瘦长年轻人进来」「为首的冷着脸」开了双弟子，后文 L13 突然单称「弟子从怀中取出玉盒」，L15 又「那弟子神色不变」回头指这个没建立的「弟子」。用户问：「弟子从怀中取出玉盒 弟子是谁」。同类裸称还散见 L27/L31/L35/L51（「玄阴宗弟子冷笑」「弟子顿了顿」「弟子脸色变了变」「弟子脸上掠过喜色」），均指前文已建立的「为首的」那一名，却只写裸「弟子」。
- **现象**：场景里出现 ≥2 个同类角色（两个弟子 / 一伙散修 / 几位长老）后，叙事用裸角色通称（「弟子」「青年」「那人」「汉子」）去指代其中某一个，却不说明是哪个，读者被迫回找"到底说的谁"，甚至误以为是新登场的人。与 #036（专名提前掉出）不同：本例不是名字问题，是**通称在多个候选中无明确先行词**。
- **根因**：AI 在单一焦点视角下默认"我说的就是刚在动作的那个"，省略了区分标记；但读者只有文字，多个同类并列时裸称即歧义。
- **错误写法（片段 ≤150 字）**：
  > 两个玄阴宗弟子押着年轻人进来。为首的冷着脸报了罪状。弟子从怀中取出玉盒，盒里三株草。
  > （"弟子"是俩里的哪个？读者无法定位，像是凭空又来一个）
- **正确写法（多候选时，每个指代都锚定到已建立的特定个体）**：
  > 两个玄阴宗弟子押着年轻人进来。为首的冷着脸报了罪状。**为首的**从怀中取出玉盒，盒里三株草。
  > （全程用已建立的「为首的」统一指代那名主动管事的弟子；另一名仅在需要时以「他身后同伴/另一名」点出）
- **可固化规则**：
  - 场景引入 ≥2 个同类角色时，立刻给一个**可复用的区分标签**（为首的 / 年长的 / 紫衣那名 / 押左的），后文一律用该标签指代，不退回裸通称。
  - 写完一段后 grep 通称（弟子 / 青年 / 那人 / 汉子 / 老者），逐个自问"这句的先行词唯一吗？"有多个同类候选却无标签的，补标签。
  - 若确实需要切换指代对象，显式写出切换（「另一名弟子」「他身旁那名」），勿靠读者脑补。
  - 与 #036 关系：#036 治「专名提前用」，本例治「通称在多人中无着落」；两者都是"指代无明确先行词"，叠加覆盖 ch9 此类全部指代表述。

## 案例 #044 — 角色态度（嚣张/卑微/冷漠）不能靠病句体现（语气≠病句）

- **来源**：2026-07-19 ch9 L15。弟子台词原句：「宗主的意思，既是要走你这『天道裁』，便让你裁。裁完，我们带走人。」用户反馈：「看不懂, 嚣张跋扈也得讲人话啊」——点破两层：①关联词错配（"既是…便…"应为"既然…就 / 既…就"）；②成分残缺（"走你这天道裁"缺"的过场/程序"依托，单看费解）；③态度（嚣张）被当成病句遮羞布。
- **现象**：为了让角色显得嚣张/卑微/冷漠，写出关联词错配、成分残缺、逻辑断的连接句，读者读不通。作者误以为"态度到位就行"，实则态度靠**措辞与句式**体现，不靠牺牲语法。
- **根因**：把"角色语气"和"句子通顺"混为一谈——嚣张可以用短促命令句、不解释、不客气用语表现，但每个分句自身语法必须成立、读者一眼读懂。
- **错误写法（片段 ≤150 字）**：
  > 「宗主的意思，既是要走你这『天道裁』，便让你裁。裁完，我们带走人。」
  > （"既是…便…"关联词错配；"走你这天道裁"缺"的过场"依托，费解；"便让你裁"的"让"有歧义）
- **正确写法（态度通过短句与用语呈现，句子通顺，且不与前文复读）**：
  > 「既走你这天道裁，你裁。裁完，我们带人走。」
  > （承接 L7 已说的"宗主令…走天道裁的过场"，不重述"宗主令/押来"；"既走你这天道裁"只做连接钩子，新信息落在"你裁…裁完带走"；短促命令句显居高临下，通顺无歧义）
  > 注：初版"宗主有令，人押来走你这天道裁"虽通顺，但与 L7 信息重复——治完本例（语气≠病句）还要过 #001 删复读，确认不与前文撞车。
- **可固化规则**：
  - 角色态度（嚣张/卑微/冷漠/谄媚）用**措辞、句式长短、敬语与否、标点**表现，不得用病句表现。写完态度句自问：`去掉'态度'这层意图，这句话语法还成立、读者还读得懂吗？` 不成立就改通顺，态度另找合规出口。
  - 关联词配对检查：既→就（不"便"）、既然→就、虽然→但是、如果→就；不混搭。
  - 动宾/介宾残缺检查：动词/介词后缺宾语或依托（如"走你这天道裁"缺"的过场"）要补全或换说法。
  - 与 #001（三删三加·说人话）、#002（设定一致·不杜撰归因）关系：本例治"为语气牺牲可读性"，是 #001"说人话"在角色台词维度的强化细则。

## 案例 #056 — 问答语义层须一致（概念偷换 / 亲历者降格）

- **来源**：2026-07-21《告天》ch10。庭审焦点是「母族外血不录、铺面该收」，陈言论证「验血脉证真林家血脉、铺面归她」。林小月却问「先生，我、我真是我爹的女儿？」用户：「这个问题看不懂」。
- **现象**：① 角色在关键节点的台词，用更戏剧的疑问**偷换**前文已建立的冲突焦点（把「族谱归属/铺面归属」偷成「亲生关系」）；② 亲历者事后用「结果真发生了吗」式确认句，把自己降格成不知情旁观者。
- **根因**：作者为求戏剧性或图省事，让角色问/说脱离其真实处境的话；读者需反推两层才懂，或与前文设定打架。
- **错误写法**：
  > 林小月（全程在场亲见验线、亲闻林鹤认输）：「先生，我、我真是我爹的女儿？」（偷换焦点 + 亲历者降格）
- **正确写法**：
  > 林小月（得意）：「先生，他方才那副熊样！这会儿灰溜溜认了栽，活该！我爹的铺面，到底还是我的了！」（贴合庭审结论，不偷换、不降格）
- **可固化规则**：
  - 角色关键台词须与其**真实焦虑/处境**一致，不可用更大/更戏剧的疑问偷换前文已建立的冲突焦点；问答须同一语义层，答句要接得住问句。
  - 全程在场的当事人，事后反应应是亲历后的情绪/后续推进，不得用「结果真发生了吗」式确认句把他降格成旁观者（与 #038 视角未卜先知、#029 认知越界互补）。

## #063 叙事概念主宾须自洽（谁欠谁 / 谁对谁做了什么）
- **现象**：写多案件罗列或对话时，把"债主/债户""原告/被告"等关系词的身份用反，且动作主宾拧巴。如 ch11 初稿：「债主赖账，反咬债户『未借』」——债主是债权人，赖账通常是债务人行为；"反咬债户未借"动作主语宾语全乱，同段后文又写"债主哑"，而配角台词说"欠债不还"，叙述者称呼与被审人真实身份自相矛盾。读者直接懵"到底谁欠谁"。
- **根因**：AI 长程生成多案件时，对"谁欠谁"的关系代词未与证据链对齐，随手写称谓；同一段内叙述者称呼与被审人身份脱节。
- **错误写法**：「债主赖账，反咬债户『未借』」「债主哑……债主面如死灰」（实际被审的是欠钱的人）
- **正确写法**：先定"谁欠谁、谁在审"再下笔，统一真实身份——「债户赖账，立誓反称『未借』」「那债户哑……那债户面如死灰」；且叙述者称呼、被审人台词（"你说未借"）、旁人总结（"欠债不还"）三者指向同一人。
- **可固化规则**：凡写债务/纠纷案，动笔前先锚定"谁欠谁、此刻谁在被审"；同一段落内叙述者对被审人的称呼须与证据（收讫誓/亏欠痕/借契）及旁人台词指向同一主体，禁止主宾颠倒或中途换词制造内部矛盾。
