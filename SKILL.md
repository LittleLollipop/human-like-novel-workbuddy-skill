# 仿人类小说创作

从根源上解决AI生成痕迹问题，让AI写出具有"人感"的小说。

## 问题诊断

AI生成文本有明显痕迹的四大原因：

1. **大量重复描写**：AI倾向于反复强调同一个点（刻意点题）
2. **节奏高度统一**：段落长度、句式分布过于均匀，没有变化
3. **用词单调**：高频词集中，缺乏多样性，情绪与用词脱节
4. **过于精确的数字**：AI习惯使用精确数字（37秒、142步），人类更常用模糊表达

本技能从**根源**上解决这些问题，而不是在生成后修补。

## 工作流程

### 阶段一：设定构建（正文创作前必须完成）

#### 步骤1：初始化项目

创建项目结构，加载模板。

**目录结构**：
- `settings/`：世界观、角色、地理、规则设定
- `settings/examples/`：**设定点写作范例库**（AI痕迹根治核心）
- `settings/characters/`：角色设定 + **情绪词库**
- `chapters/plans/`：**章节创作计划**（AI痕迹根治核心）
- `tracker/`：伏笔、冲突、反馈追踪

#### 步骤2：录入设定 + 构建写作范例库

**2.1 基础设定录入**

支持的类型：`world`（世界观）/ `character`（角色）/ `rule`（规则）/ `geography`（地理）

**2.2 构建设定点写作范例库（核心）**

每个主要设定点必须提供 **3-5个** 不同角度的写作范例。

**为什么需要范例库？**
AI最大的问题之一是"直白点题"——反复用相同的方式强调同一个主题。范例库提供多种不同的点题方式，AI从库中选一个改编使用，**禁止自创点题方式，禁止重复使用同一范例**。

**范例角度要求**（至少3种）：
- 直接描写
- 对话体现
- 内心独白
- 环境映射
- 他人视角

**范例文件格式**（`settings/examples/<设定点名>.md`）：

```markdown
# 设定点：<名称>

## 角度一：直接描写
<150字以内的写作范例>

## 角度二：对话体现
<150字以内的写作范例>

## 角度三：内心独白
<150字以内的写作范例>

## 角度四：环境映射
<150字以内的写作范例>

## 角度五：他人视角
<150字以内的写作范例>
```

**使用规则**：
- 正文创作时如需"点题"，**必须从范例库中选取一个范例进行改编**
- 禁止直白重复范例原文
- 禁止全文重复使用同一范例（每个范例最多用2次，且必须间隔至少10章）

#### 步骤3：构建角色情绪词库（人感核心）

**为什么需要情绪词库？**

AI写作"平淡"的根本原因是：**用词没有情绪偏向**。

人类作者会本能地让词语携带角色情绪：
- 愤怒时 → 用词具有攻击性（动词：砸、撕、掐；形容词：尖锐、滚烫）
- 自卑时 → 用词懦弱（动词：缩、躲、低声；形容词：灰暗、微小）
- 悲伤时 → 用词沉重（动词：瘫、滑、闭；句式：长句、停顿、省略号）

情绪词库是用词人感的核心工具。

**文件格式**（`settings/characters/<角色名>-emotion.md`）：

```markdown
# 角色：<名字>

## 基本信息
- 身份/背景：<角色身份，如"克隆人战士"、"古代剑修">
- 教育背景：<角色受过什么教育，如"自学古典文学"、"只接受过军事训练">
- 性格特征：<核心性格>

## 情绪状态 → 用词特征映射

### 愤怒
- 动词倾向：砸、撕、掐、吼、跺
- 形容词倾向：尖锐、滚烫、刺耳、猩红
- 句式倾向：短句、断句、反问句
- 禁忌：平静叙述、长篇解释、委婉表达

### 自卑/怯懦
- 动词倾向：缩、躲、低声、后退、垂下
- 形容词倾向：灰暗、微小、模糊、瑟缩
- 句式倾向：不确定语气（"可能"/"也许"）、被动句
- 禁忌：果断决定、主动出击的描写

### 悲伤/绝望
- 动词倾向：瘫、滑、闭、颤抖
- 形容词倾向：冰冷、空洞、沉重、灰白
- 句式倾向：长句、停顿、省略号
- 禁忌：轻快节奏、明亮词汇

### 兴奋/喜悦
- 动词倾向：跳、冲、抓、亮
- 形容词倾向：滚烫、耀眼、清脆、鲜活
- 句式倾向：短句、感叹句、排比
- 禁忌：冷静分析、负面词汇

### 冷静/算计
- 动词倾向：敲、扫、计算、排列
- 形容词倾向：清晰、冰冷、精确、有序
- 句式倾向：陈述句、逻辑连接词
- 禁忌：情绪化表达、夸张修辞

### 紧张/恐惧
- 动词倾向：吞、缩、僵、抖
- 形容词倾向：冰冷、尖锐、狭窄
- 句式倾向：短句、断句、重复词
- 禁忌：从容不迫的描写

### 疲惫/麻木
- 动词倾向：垂、拖、靠、闭
- 形容词倾向：灰、沉、钝、慢
- 句式倾向：长句、缓慢节奏
- 禁忌：激烈动作、高昂情绪

## 流行文化熟悉度（非严肃文学适用）

**互联网使用习惯**：

- 熟悉程度：深度冲浪 / 偶尔刷刷 / 完全不懂
- 年龄层：Z世代 / 千禧 / 中年 / 老年（影响梗的使用）

**引用风格偏好**：
- 适合：网络热梗、流行语、梗图描述、二次元引用
- 不适合：过于严肃的场合、与角色身份不符的场景
- 使用频率：高频（每章多次）/ 中频（每章1-2次）/ 低频（偶尔）

**搜索关键词库**（创作时用来搜索热梗的关键词）：
- 幽默场景：最近流行搞笑梗、网络热梗

**使用规则**：
- 必须符合角色身份（古代角色不会用现代梗）
- 必须符合场景氛围（悲伤场景不适合搞笑梗）
- 必须自然（不能为了玩梗而玩梗）
```

**使用规则**：
1. 每个主要角色必须建立情绪词库
2. 每章创作前，明确每个角色在本章的情绪状态
3. 创作时，对照情绪词库选择用词和句式
4. 角色情绪变化时，用词必须同步变化
5. 遇到引用触发点时，对照"引用偏好"搜索合适内容
6. 遇到流行文化引用触发点时，对照"流行文化熟悉度"搜索合适内容

### 阶段二：章节计划（正文创作前必须完成）

#### 步骤4：创建章节详细计划

**强制规则**：在写正文之前，必须为每章生成详细的创作计划。

**为什么需要章节计划？**

AI生成的正文往往"平淡"、"匀速"、"没有重点"，因为AI不知道本章应该"怎么写"。章节计划通过**三要素**给AI明确的约束：

**计划三要素（缺一不可）**：

```markdown
# 第X章 创作计划

## 基本信息
- 章节标题：
- 所属卷：
- 预计字数：

---

## ★ 三要素约束（创作时必须严格遵守）

### 要素①：行文风格
控制本章的节奏和句式分布。

**节奏分布**（必须明确指定一种）：
- [快-慢-快]：开场冲突 → 对话/反思 → 再次冲突/转折
- [慢-快-慢]：氛围铺垫 → 高潮行动 → 余韵收尾
- [匀速紧张]：全程保持紧凑
- [舒缓铺陈]：慢节奏，重氛围

**句式混合要求**：
- 单句段落：至少 ___ 处（建议：动作场面≥5处，对话场面≥3处）
- 长段落（>200字）：不超过 ___ 处
- 对话占比：约 ___ %（根据创作重点调整）

**章节开头方式**（必须与前章不同）：
- 对话切入 / 动作切入 / 感官细节 / 内心独白 / 场景描写
- 检查前章开头方式，确保不重复

**章节结尾方式**（禁止总结性点题）：
- 悬念留白 / 情感余韵 / 行动决定 / 对话截断
- 禁止：在章节末尾添加"这一刻，他明白了..."之类的总结句

### 要素②：创作重点
控制本章的内容比重和偏向。

**本章核心偏向**（单选主倾向，可配次要倾向）：
- [ ] 感情戏（角色关系推进，重对话和内心戏）
- [ ] 剧情推进（事件发生/转折，重动作和因果）
- [ ] 世界背景介绍（设定披露，重环境和规则描写）
- [ ] 环境描写（氛围营造，重感官细节）
- [ ] 动作/战斗场面（重节奏和动词）
- [ ] 内心戏（角色成长/反思，重情绪和独白）
- [ ] 对话戏（信息交换/性格展现，重声线差异）

**重点说明**（2-3句话描述本章最想达成的效果）：
<填写>

**内容比重建议**：
- 根据创作重点，明确各部分字数占比
- 例：对话40% + 动作30% + 内心戏20% + 环境10%

### 要素③：情绪基调（人感核心）
控制本章的用词选择，让文字携带情绪。

**本章整体情绪**：
- 主情绪：[压抑/亢奋/悲伤/紧张/轻松/绝望/温暖/...]
- 情绪变化曲线：起始情绪 → 转折事件 → 结尾情绪

**角色情绪状态**（影响用词选择）：
- 主角：<情绪状态> → 参照 `characters/<主角>-emotion.md` 的对应词库
- 配角A：<情绪状态> → 同理
- 配角B：<情绪状态> → 同理

**用词约束**：
- 本章所有叙述和对话必须符合上列情绪状态
- 禁止情绪与用词背离（如角色愤怒但用词平和、犹豫）
- 叙述者的语气也必须与章节整体情绪协调

## ★ 负面约束（禁止事项，创作时必须逐条检查）

- [ ] **禁止直白点题**：主题必须通过情节/动作/对话自然体现，不得在文中出现"这说明..."、"他意识到..."之类的直白总结
- [ ] **禁止精确数字**：除严谨必要场景（如科技设定、战斗倒计时）外，所有时间/距离/数量必须用模糊表达：
  - ✅ 片刻、数十米、百来步、半刻钟、三两下、片刻之后
  - ❌ 37秒、142步、8.5米、23分钟、正好3次
  - 例外场景每章不超过3处，且必须在计划中注明
- [ ] **禁止重复前章用词**：从前面3章提取高频形容词和动词，本章强制避开
- [ ] **禁止连续3段使用相同句式**：每段结束后检查下一段的句式是否变化
- [ ] **禁止节奏均匀分布**：段落长度必须有意混合短段（1-2句）和长段（5句以上），不得全部是3-4句的中等段落
```

## 关键原则（必须遵守）

1. **设定不全，不写正文**：正文创作前必须完成基础设定
2. **范例库不全，不写正文**：主要设定点必须有写作范例
3. **情绪词库不全，不写正文**：主要角色必须有情绪词库
4. **计划不全，不写正文**：章节计划三要素（行文风格/创作重点/情绪）缺一不可
5. **情绪词库是用词人感的核心**：每次更新角色设定时必须同步更新情绪词库
6. **设定点范例是点题的唯一合法方式**：禁止自创点题方式，必须从范例库选取改编
7. **负面约束优先于正面要求**：当"写什么"和"不写什么"冲突时，优先遵守"不写什么"
8. **模糊数字优先**：默认使用模糊数字，精确数字需要正当理由
9. **情绪必须贯穿全文**：从章节计划到正文创作到自检，情绪基调必须全程可追溯
10. **引用触发机制**：不在本地建素材库，创作时实时搜索，宁缺毋滥
11. **引用必须符合角色背景**：引用前必须检查角色的引用偏好和知识背景
12. **流行文化引用必须符合角色身份**：古代角色不会用现代梗，严肃角色不会随意玩梗
13. **流行文化引用必须符合场景氛围**：悲伤场景不适合搞笑梗，战斗场景不适合玩梗
14. **流行文化引用必须自然**：不能为了玩梗而玩梗，宁缺毋滥
